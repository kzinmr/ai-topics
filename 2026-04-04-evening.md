### 3. Simon Willison の「Dark Factory」- レベル 5 の自律的ソフトウェア開発
- **5 レベルモデル** (Dan Shapiro 提案):
  - **レベル 1**: AI 支援コーディング (Copilot 等)
  - **レベル 2**: AI がコード生成、人間がレビュー
  - **レベル 3**: AI がコード生成・テスト、人間が承認
  - **レベル 4**: AI が開発の大部分、人間が仕様設計
  - **レベル 5 (Dark Factory)**: AI が全てを自律実行、人間は監視のみ
- **転換点**: 2025 年 11 月、「mostly works」→「actually works」
- **実例**: StrongDM では小チームが仕様・テストシナリオを設計、AI が全コーディング・QA を実行
- **人間の役割変化**: ソフトウェア構築から、構築システムを設計・監視する役割へ
- **リスク**:
  - **ミッドキャリアエンジニア**が最も危険 (ジュニアではない)
  - **プロンプトインジェクション**: 未解決のセキュリティ問題
  - **「AI Challenger 災害」**: 2026 年に予測

### マルチエージェントワークフロー例 (Python)

```python
import asyncio
from agent_framework import Agent, Message
from agent_framework.foundry import FoundryChatClient
from agent_framework.orchestrations import SequentialBuilder
from azure.identity import AzureCliCredential

async def main() -> None:
    # 認証 (環境変数から読み取り)
    client = FoundryChatClient(credential=AzureCliCredential())

    # 執筆エージェント
    writer = Agent(
        client=client,
        instructions="You are a concise copywriter. Provide a single, punchy marketing sentence.",
        name="writer",
    )

    # レビューエージェント
    reviewer = Agent(
        client=client,
        instructions="You are a thoughtful reviewer. Give brief feedback on the previous message.",
        name="reviewer",
    )

    # シーケンシャルワークフロー構築
    workflow = SequentialBuilder(participants=[writer, reviewer]).build()
    
    # 実行 (ストリーミング)
    outputs: list[list[Message]] = []
    async for event in workflow.run("Write a tagline for Microsoft Agent Framework 1.0.", stream=True):
        if event.type == "output":
            outputs.append(cast(list[Message], event.data))
    
    # 結果出力
    if outputs:
        for msg in outputs[-1]:
            print(f"[{msg.author_name or 'user'}]: {msg.text}")

asyncio.run(main())
```
