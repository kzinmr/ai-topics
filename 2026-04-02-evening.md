### 2. HackerNews 議論 - AI エージェントの現実と課題

### 3. Karpathy "AutoResearch" - 自律的 ML 実験の衝撃
- **21K+ GitHub stars**: 3 月リリース後、大きな反響
- **機能**: AI コーディングエージェントが単一 GPU で自律的に ML 実験を実行
- **サイクル**: 提案 → トレーニング → 評価 → 改善コミット
- **Karpathy のコメント**: 「フロンティア研究を夜間に実行する'swarm agents'の種」
- **agentic engineering**: エンジニアはコードを直接書くのではなく、AI エージェントをオーケストレーション
- **2025 年 12 月以降**: 「コードを 1 行もタイピングしていない」
- **新しい「動詞」**: "manifesting" - 意図を表現、目標を分解、タスクをエージェントに割り当て、マクロレベル出力をレビュー
- **プログラミングの再構築**: 希少リソースは「flops」ではなく「tokens」（エージェントでコマンドできるトークンスループット）
- [The New Stack](https://thenewstack.io/karpathy-autonomous-experiment-loop/)

### 4. Claude Code ソースリーク続報 - 内部アーキテクチャ暴露
- **3 月 31 日/4 月 1 日**: Anthropic が「ヒューマンエラー」で 512,000 行のコードをリーク
- **判明した内容**:
  - マルチエージェントオーケストレーションシステム
  - 「Undercover Mode」（Anthropic 社名を隠す）
  - 「frustration detector」（ユーザーのフラストレーション検出）
  - 隠しコンパニオンペットシステム
  - 内部プロジェクト: Mythos, Capybara, Opus 4.7, Sonnet 4.8
  - KAIROS (Always-On Claude), BUDDY (コンパニオン)
- **セキュリティ懸念**: AI 安全を強調する企業の運用セキュリティへの疑問
- **オープンソース抽出**: リークされたマルチエージェントシステムが任意 LLM 対応のオープンソースフレームワークとして抽出
- **法的対応**: Anthropic が 8,000 件以上の著作権削除通知を発行

### 5. GitHub Copilot データポリシー変更 - 4/24 が期限
- **2026 年 4 月 24 日** より、Free/Pro/Pro+ プランのインタラクションデータが AI モデルトレーニングに使用される
- **対象データ**:
  - 受け入れ/修正された出力
  - Copilot に送信された入力（コードスニペット含む）
  - カーソル位置のコードコンテキスト
  - 作成したコメント・ドキュメント
  - ファイル名、リポジトリ構造、ナビゲーションパターン
  - Copilot 機能（チャット、インライン提案）とのインタラクション
  - 提案へのフィードバック（サムズアップ/ダウン）
- **影響外**: Business/Enterprise（既存 DPA によりトレーニング使用なし）、学生・教員
- **オプトアウト方法**:
  1. GitHub Account Settings → Copilot
  2. Privacy セクションで「Allow GitHub to use my data for AI model training」を無効化
- **注意**: 以前オプトアウト済みの設定は保持されるが、明示的にオプトインしない限りトレーニングに使用されない
- [GitHub Blog](https://github.blog/news-insights/company-news/updates-to-github-copilot-interaction-data-usage-policy/)
