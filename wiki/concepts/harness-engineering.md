# Harness Engineering

> **Note:** This page is a stub. See [[harness-engineering]] for the full concept.

## Summary

Harness Engineering is the environmental design philosophy where "Agent = Model + Harness". It focuses on designing the environment (harness) in which agents operate autonomously to write code, run tests, and create PRs.

Key figures: Ryan Lopopolo (OpenAI), Simon Willison (Agentic Engineering), Anthropic (AI Agent Engineering).
