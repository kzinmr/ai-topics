# Monty Sandbox

Monty is a sandboxing project built on Pydantic, designed to provide safe, structured execution environments for AI agents. It extends Pydantic's validation and serialization capabilities to create controlled sandbox contexts.

## Key Concepts

- **Structured Isolation**: Uses Pydantic models to define and enforce sandbox boundaries
- **Tool Execution**: Provides a framework for defining and running agent tools in isolated contexts
- **State Management**: Handles serialization and state persistence across sandbox boundaries
