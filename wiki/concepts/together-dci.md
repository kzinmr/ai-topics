# Together DCI (Deploy and Inference)

Together AI's Deploy and Inference platform enables deploying any model from HuggingFace with optimized inference.

## Key Features
- **Serverless deployment**: Deploy models without managing infrastructure
- **HuggingFace integration**: Direct deployment from HuggingFace model hub
- **Optimized inference**: Built-in optimizations for efficient model serving
- **Multi-model support**: Run different models simultaneously

## Use Cases
- Rapid prototyping with different open-source models
- Production inference at scale
- Model comparison and evaluation
- Custom model deployment pipelines
