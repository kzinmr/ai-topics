# Agentic Manual Testing

> **Note:** This page is a stub and needs expansion. See [[concepts/harness-engineering/agentic-workflows/agentic-manual-testing.md]] for the full content.

## Summary

Agentic Manual Testing combines human manual testing with AI agent capabilities:

- AI agents assist in generating test cases
- Agents can reproduce and verify bugs
- Human testers focus on exploratory testing and edge cases
- Agents handle repetitive verification tasks

## Key Principles

From Simon Willison's approach:
- Use agents to automate repetitive manual verification
- Keep humans in the loop for judgment-heavy testing
- Let agents handle test execution and reproduction
