# Agentic Engineering (Simon Willison)

Simon Willison discussed "agentic engineering" on Lenny's Podcast — the practice of using AI agents as software engineering collaborators rather than just code completion tools.

## Key Concepts

### Agentic Engineering
- AI agents as **collaborators** in the software development process
- Beyond code completion — agents that can plan, execute, and verify tasks
- Human-in-the-loop workflows where agents handle complex multi-step work

### Practical Applications
- Code generation and refactoring
- Testing and debugging
- Documentation and knowledge management
- System design and architecture review

### Workflow Integration
- Agents as persistent team members rather than one-off tools
- Iterative development with agent feedback loops
- Quality assurance through agent-driven testing
