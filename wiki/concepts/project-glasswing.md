# Project Glasswing

Anthropic's initiative to use [[claude-mythos]] for defensive security research — finding and patching vulnerabilities in critical software.

## Overview

Instead of releasing Mythos publicly, Anthropic created Project Glasswing to harness the model's vulnerability discovery capabilities for beneficial purposes.

## Commitment

| Resource | Amount |
|----------|--------|
| Model usage credits | $100M |
| Donations to open-source security orgs | $4M |

## Participants

12 companies have been granted preview access to Mythos under this program.

## Discoveries

Mythos has already identified long-standing security bugs in critical software:
- **OpenBSD**: 27-year-old vulnerability
- **FFmpeg**: 16-year-old vulnerability
- Additional critical software projects

## Significance

Project Glasswing represents a novel approach to AI safety — rather than withholding a powerful model entirely, redirecting its capabilities toward defensive applications under controlled access.

> [!note] AI Safety Implication
> This is a case study in responsible AI deployment: a model too dangerous for general release being used productively for security research.
