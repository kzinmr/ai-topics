# Capabilities-Based Security

> **Note:** This page is a stub and needs expansion.

## Summary

Capabilities-Based Security is a security philosophy where:

- **Default: Zero Access** — Agents start with no permissions
- **Explicit Grant** — Capabilities must be explicitly granted
- **Minimal Privileges** — Only grant what's strictly necessary
- **Revocable** — Capabilities can be withdrawn

## Monty Philosophy

The concept is strongly associated with Monty (minimal secure Python interpreter in Rust):

> "Start from zero access, explicitly grant capabilities"
