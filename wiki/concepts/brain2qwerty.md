# Brain2Qwerty (Meta v2)

## Overview

Brain2Qwerty v2 is Meta's non-invasive brain-to-text decoding system that translates raw electroencephalography (EEG) brain signals into typed text in real time. The system represents a significant milestone in non-invasive [[concepts/deep-learning|brain-computer interface]] technology, achieving approximately **61% accuracy** in decoding sentences directly from brain activity without requiring surgical implantation.

## Technical Details

- **Modality**: Non-invasive EEG-based signal acquisition
- **Task**: Real-time sentence decoding from raw brain signals
- **Accuracy**: ~61% word-level decoding accuracy
- **Approach**: Deep learning models trained on EEG recordings of users typing natural sentences

Unlike invasive brain-computer interfaces (BCIs) that require implanted electrodes, Brain2Qwerty operates entirely through scalp-mounted sensors, making it practical for broader consumer and clinical applications without surgical risk.

## Coding Agents and Auto Research Workflow

A key innovation in Brain2Qwerty v2 came from applying coding agent-powered **Auto Research** workflows to improve the system's word error rate. The [[concepts/auto-research|Auto Research]] methodology — an AI agent framework for automating the research pipeline — was used to autonomously design experiments, write and run code, and iterate on decoding architectures. This agent-driven approach systematically improved decoding performance, demonstrating how [[concepts/coding-agents|coding agents]] can accelerate neuroscience and BCI research.

## Significance

Brain2Qwerty v2 is a landmark achievement in non-invasive neurotechnology for several reasons:

1. **Non-invasive milestone**: Proves that meaningful brain-to-text communication is achievable without brain surgery, addressing the primary barrier to BCI adoption.
2. **Real-time performance**: Demonstrates practical real-time decoding speeds suitable for assistive communication.
3. **Agent-accelerated research**: Validates the use of autonomous coding agents to drive scientific discovery in neurotechnology.
4. **Accessibility potential**: Opens the door to affordable, low-risk communication aids for individuals with motor impairments such as ALS or locked-in syndrome.
