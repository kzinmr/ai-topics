# Vibe Coding

> **Note:** This page is a stub and needs expansion. See [[harness-engineering/agentic-workflows/vibe-coding]] for the full content.

## Summary

Vibe Coding is a development approach characterized by:
- Loose, conversational prompts to AI coding agents
- Minimal oversight during code generation
- Accepting AI-generated code with "vibe" rather than strict review
- High velocity but potentially lower quality output
