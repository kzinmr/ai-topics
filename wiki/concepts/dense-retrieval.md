# Dense Retrieval

**Dense retrieval** is an information retrieval paradigm that uses learned embedding vectors (dense representations) to encode query and document semantics, then performs nearest-neighbor search over the embedding space. Unlike sparse retrieval (e.g., BM25) which relies on exact term matching, dense retrieval captures semantic similarity even without lexical overlap.
