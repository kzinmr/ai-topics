# Linear Walkthroughs

> See: [harness-engineering/agentic-workflows/linear-walkthroughs.md](./harness-engineering/agentic-workflows/linear-walkthroughs.md)

Linear walkthroughs are structured, step-by-step guides that walk users through processes in a sequential, predictable manner. In agentic contexts, they inform how agents should structure task completion.
