# Reinforcement Learning

Reinforcement Learning (RL) is a paradigm of machine learning where agents learn to make decisions by taking actions in an environment to maximize cumulative reward signals. RL is foundational to training advanced AI systems including language models via RLHF.

## Key Concepts

- **Agent**: The entity taking actions in the environment
- **Environment**: The context in which the agent operates
- **Reward Signal**: Feedback that guides the agent's learning
- **Policy**: The strategy the agent uses to select actions
- **Value Function**: Estimates expected cumulative reward from a given state

## Modern AI Applications

- **RLHF**: Reinforcement Learning from Human Feedback for LLM alignment
- **AlphaCode/AlphaCode 2**: Competitive programming via RL
- **AlphaGo/AlphaZero**: Game playing and general RL achievements
- **Self-play**: Agents improving by competing against themselves
