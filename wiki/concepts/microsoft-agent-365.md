# Microsoft Agent 365

> **Launched:** May 1, 2026 | **Pricing:** $15/user/month | **Type:** Enterprise AI Agent Governance Control Plane

Microsoft Agent 365 is a dedicated **governance and security control plane** for enterprise AI agents. It is a separate product from Microsoft 365 Copilot, positioned as the security layer that sits **above** the AI features in Office apps.

## What It Does

Agent 365 gives enterprise IT teams visibility and control over autonomous AI systems running across their organization. It manages agents built on:
- **Microsoft Foundry** — AI development platform
- **Copilot Studio** — Low-code agent builder
- **Third-party agents** — External AI systems operating within the enterprise

## Why It Matters

Agent 365 addresses the critical enterprise blocker for AI agent adoption: **governance and oversight**. While models like GPT-5.5 and Claude Opus 4.7 can now autonomously execute complex tasks, enterprises have been reluctant to deploy them without:
1. Visibility into what agents are doing
2. Control over what agents can access
3. Auditability of agent actions
4. Security policies that span multiple agent platforms

This is Microsoft's direct response to the shift from "can the AI do it?" to **"how do we govern it?"** — the new bottleneck in enterprise AI.

## Relationship to Wave 3

Microsoft's "Wave 3" (March 9, 2026) integrated AI into Office apps. Agent 365 is the **security and governance layer** above those apps — a recognition that AI features deployed without oversight create unacceptable enterprise risk.

## Target Audience

Enterprise IT and security teams. Not designed for individual users or small businesses.
