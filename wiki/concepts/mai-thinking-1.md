# MAI-Thinking-1

MAI-Thinking-1 is a powerful reasoning model developed from scratch by Microsoft AI (MAI) that competes with models of similar size on STEM reasoning and coding.

## Key Ideas
- **Hill-Climbing Machine**: A process of treating model development as a system-level optimization problem, using reinforcement learning for rapid improvement.
- **Reinforcement Learning Recipe**: A robust RL framework optimized for sustained log-linear performance improvement over thousands of steps.
- **Scaling-Focused Framework**: Emphasizing empirically-driven iterative improvements to architecture and data.
