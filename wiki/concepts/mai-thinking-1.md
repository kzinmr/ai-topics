# MAI-Thinking-1

MAI-Thinking-1 is a 35B active / 1T total parameter Mixture of Experts (MoE) reasoning model developed by Microsoft AI.

## Key Features
- **From-Scratch Training**: Trained on clean, enterprise-grade data without distillation from third-party models.
- **Reinforcement Learning**: Uses a robust RL recipe to sustain log-linear performance improvement over thousands of steps.
- **Performance**: Highly competitive on STEM reasoning and coding tasks (e.g., SWE-Bench Pro, AIME 2025).
- **Hill-Climbing Machine**: Developed using a framework that treats model development as a system-level optimization problem.
