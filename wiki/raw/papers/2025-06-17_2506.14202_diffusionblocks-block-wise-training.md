# DiffusionBlocks: Block-wise Neural Network Training via Diffusion Interpretation

**Authors:** Makoto Shing¹, Masanori Koyama², Takuya Akiba¹
¹Sakana AI, ²The University of Tokyo
**Venue:** ICLR 2026
**Code:** https://github.com/SakanaAI/DiffusionBlocks

## Abstract

End-to-end backpropagation requires storing activations throughout all layers, creating memory bottlenecks that limit model scalability. **DiffusionBlocks** transforms transformer-based networks into genuinely independent trainable blocks by interpreting residual connections as Euler discretization of a continuous-time diffusion process. Each block learns to denoise within an assigned noise-level range, requiring gradients for only one block at a time, reducing memory requirements proportionally to the number of blocks. Experiments on vision, diffusion, autoregressive, recurrent-depth, and masked diffusion transformers show competitive performance with end-to-end training.

## Key Insight

> "Our key insight leverages the fact that residual connections naturally correspond to updates in a dynamical system. With minimal modifications to this system, we can convert the updates to those of a denoising process, where each block can be learned independently by leveraging the score matching objective."

Residual connections: z_ℓ = z_{ℓ-1} + f_{θ_ℓ}(z_{ℓ-1}) are seen as a single Euler step of the reverse diffusion ODE: dz_σ/dσ = -σ ∇_z log p_σ(z_σ), where the denoiser D_θ is trained with score matching loss.

## Method: 3-Step Conversion

### Step 1: Block Partitioning
Partition L layers into B blocks. f̄_{θ_b} := f_{θ_{ℓ_b}} ∘ ... ∘ f_{θ_{ℓ_{b-1}+1}}

### Step 2: Noise Range Assignment
Choose noise distribution p_noise (log-normal recommended) and range [σ_min, σ_max]. Partition into B intervals using equi-probability partitioning.

### Step 3: Noise Conditioning
Extend block input to (x, z_σ) where z_σ = y + σϵ. Incorporate noise-level conditioning via AdaLN.

## Block-Independent Training

Each block b trained independently with:
L_b(θ_b) = E_{(x,y)~p_data, σ~p_noise^(b), ϵ~N(0,I)} [ w(σ) · Loss( f̄_{θ_b|σ}(x, y+σϵ), y ) ]

- p_noise^(b) is the noise distribution restricted to [σ_b, σ_{b-1}] and renormalized
- Weighting w(σ) follows EDM (Karras et al.)
- Only L/B layers require gradient computation → B× memory reduction during training

## Inference
Starting from z_0 ~ N(0, σ_max² I), iterate through noise levels σ_i, select block covering that σ_i, denoise, apply Euler step. Typically number of denoising steps T = B.

## Extensions

| Architecture | Adaptation |
|---|---|
| ViT (classification) | Noise added to class label embeddings; blocks denoise label conditioned on image patches |
| Diffusion models (image generation) | Natural fit; blocks directly act as denoisers for assigned noise ranges |
| Autoregressive language models | Noise added in embedding space; causal consistency maintained via concatenation |
| Recurrent-depth transformers | Block recurrence naturally fits iterative denoising |
| Masked diffusion transformers | Combined with masked training objectives |

## Key Results

- Memory reduction proportional to number of blocks (B×)
- Competitive performance with end-to-end training across architectures
- Successfully scales block-wise training beyond small-scale classification
- Equi-probability partitioning outperforms uniform partitioning
- Log-normal noise distribution (EDM defaults) works best
