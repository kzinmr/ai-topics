# DeepSeek-R1: Incentivizing Reasoning Capability in LLMs via Reinforcement Learning

## Summary

DeepSeek-R1 demonstrates that advanced reasoning capabilities can emerge from **pure Reinforcement Learning (RL)** without reliance on human-annotated Chain-of-Thought (CoT) demonstrations. Published in Nature (2025).

## Key Models

1. **DeepSeek-R1-Zero**: Trained via pure RL from base model (DeepSeek-V3-Base). No SFT. Emerges self-reflection, verification, "aha moments."
2. **DeepSeek-R1**: Production model using multi-stage pipeline: Cold Start SFT → Reasoning RL → Rejection Sampling SFT → General RL.
3. **Distilled Models**: Reasoning patterns transferred to Qwen-1.5B/7B/14B/32B and Llama-8B/70B via SFT on R1 outputs.

## Training Algorithm: GRPO (Group Relative Policy Optimization)

Eliminates the need for a critic model by comparing relative rewards within a group:
$$A_i = \frac{r_i - \text{mean}(\{r_1, r_2, \dots, r_G\})}{\text{std}(\{r_1, r_2, \dots, r_G\})}$$

Rule-based rewards (not neural reward models) to prevent reward hacking:
- Accuracy Rewards: Verifiable correctness (math ground truth, compiler results)
- Format Rewards: Thinking process between `think` tags

## Multi-Stage Pipeline (DeepSeek-R1)

1. **Cold Start** (~few thousand long CoT examples) → readable reasoning baseline
2. **Reasoning RL** → large-scale RL on math, code, logic
3. **Rejection Sampling + SFT** → 800k samples (600k reasoning + 200k general)
4. **General RL** → helpfulness + harmlessness alignment

## Distillation Results

| Student Model | AIME 2024 | MATH-500 |
|---------------|-----------|----------|
| Qwen-1.5B | 28.9% | 83.9% |
| Qwen-7B | 55.5% | 92.8% |
| Llama-8B | 50.4% | 89.1% |
| Llama-70B | 70.0% | 94.5% |

DeepSeek-R1-Distill-Llama-70B surpasses GPT-4o and Claude-3.5-Sonnet on reasoning.

## The "Aha Moment"

During R1-Zero training, the model autonomously developed self-correction:
> "Wait, wait. Wait. That's an aha moment I can flag here. Let's reevaluate this step-by-step..."

## Training Costs

- R1-Zero: ~101K H800 GPU hours ($202K)
- R1: ~41K H800 GPU hours ($82K)
- Total: ~$294K

## Limitations

- Language mixing (English↔Chinese)
- No native tool use (search, calculator)
- Prompt sensitivity: few-shot degrades performance
- Large-scale RL not applied to software engineering tasks

## Historical Significance

First demonstration that pure RL can incentivize reasoning emergence at scale. Published in Nature — rare for an industry AI paper. GRPO introduced as a compute-efficient alternative to PPO. Distillation proved reasoning can be transferred to small models (1.5B outperforms much larger non-reasoning models).
