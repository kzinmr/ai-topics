# ECHO: Terminal Agents Learn World Models for Free

**Authors:** Vaishnavi Shrivastava, Piero Kauffmann, Ahmed Awadallah, Dimitris Papailiopoulos (Microsoft Research)

## Abstract

CLI agents emit commands, the terminal executes them, and the returned stream (stdout, errors, files, logs, traces) records the consequences. This stream is a supervision signal, but standard GRPO updates only action tokens with sparse outcome-level rewards, ignoring environment observations already in the rollout. Failed rollouts provide rich evidence about environment responses but are discarded.

ECHO (Environment Cross-entropy Hybrid Objective) adds an auxiliary cross-entropy loss on environment-observation tokens to the standard policy-gradient loss on action tokens. It reuses the same forward pass, requires no extra rollouts or teacher model, and turns every terminal response into dense, on-policy supervision.

- ECHO doubles GRPO pass@1 on TerminalBench-2.0: Qwen3-8B 2.70% → 5.17%, Qwen3-14B 5.17% → 10.79%.
- ECHO sharply reduces environment-token cross-entropy on off-policy rollouts, indicating better world modeling.
- From a base Qwen3-8B, ECHO matches expert-SFT-then-GRPO on internal evaluations without using any expert demonstrations, and closes about half the gap on TerminalBench-2.0.
- In some settings, environment prediction alone enables verifier-free adaptation on unseen OOD tasks.

> Core insight: Environment observations are not merely context for future actions, but a dense, on-policy supervision signal already present in every rollout.

## Method

ECHO is a hybrid objective:

```
L_ECHO = L_GRPO(Actions) + λ · L_env(Observations)
```

- Same forward pass and rollout as GRPO
- λ = 0.05 (constant, auto-annealing as L_env falls)
- Predicts env tokens (stdout, stderr, exit codes) — not harness warnings
- On-policy: targets come from current model's own rollouts

## Key Results

| Benchmark | Qwen3-8B (GRPO) | Qwen3-8B (ECHO) | Qwen3-14B (GRPO) | Qwen3-14B (ECHO) |
|-----------|-----------------|------------------|-------------------|-------------------|
| TerminalBench-2.0 pass@1 | 2.70% | **5.17%** | 5.17% | **10.79%** |
| Training speed | 1× | **2.3× faster** | — | faster |

## Surprising Findings

1. **Substitute for Expert SFT**: From base Qwen3-8B, ECHO recovers up to 104% of expert-SFT gain on ITD, 89% on TBLite, 50% on TB2.
2. **Verifier-free self-improvement**: With no GRPO term (only L_env), the model still improves on OOD tasks (+3.8 to +10.0 pp) when rollouts are clean and feedback is informative.
