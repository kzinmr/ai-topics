# GEPA: Reflective Prompt Evolution

**Paper:** GEPA: Reflective Prompt Evolution Can Outperform Reinforcement Learning (ICLR 2026 Oral)
**Authors:** LA Agrawal, S Tan, D Soylu, N Ziems, D Klein, M Zaharia, O Khattab (Stanford/MIT)

## Core Contribution

GEPA (Genetic-Pareto) is a sample-efficient optimizer for compound AI systems that uses three core principles:
1. **Genetic prompt evolution** — prompts breed new prompts through mutation/crossover
2. **Pareto optimality** — jointly optimizes quality vs token cost
3. **Reflective evaluation** — self-evaluation of generated prompts

## Key Results

- **6% average improvement** over GRPO (Reinforcement Learning baseline)
- **Up to 20% improvement** on some tasks
- **35x fewer rollouts** than GRPO (dramatically more sample-efficient)
- Outperforms RL-based optimization while being simpler and cheaper

## How It Works

GEPA treats prompts as evolutionary individuals:
- **Population:** Set of candidate prompts with different quality/cost tradeoffs
- **Fitness:** Pareto dominance — quality AND token efficiency
- **Mutation:** Paraphrase, specificity adjustment, format changes
- **Crossover:** Combine successful patterns from different prompts
- **Selection:** Keep Pareto-optimal front (non-dominated individuals)
