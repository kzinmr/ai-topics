# Workspace-Bench 1.0

Comprehensive benchmark evaluating AI agents on "Workspace Learning" across large-scale, heterogeneous file environments.

## Key Statistics
- 5 worker profiles, 74 file types, 20,476 files (up to 20GB)
- 388 tasks with explicit dependency graphs, 7,399 rubrics
- Best AI: 68.7% (OpenClaw + Claude Opus 4.7), Human: 80.7%, Average AI: 47.4%
- Workspace-Bench-Lite: 100-task subset (~70% cost savings)

## Core Concept
Workspace Learning: whether an agent can identify, reason over, exploit, and update explicit and implicit dependencies among heterogeneous files.

## Five Evolution Stages
L0: Single-file → L1: Multi-file homogeneous → L2: Multi-file heterogeneous explicit → L3: Multi-file implicit dependencies → L4: Adaptive (human-level)

## Key Findings
- Harness design is a major factor in performance
- Scale (20GB, 74 file types) creates retrieval challenges not captured by existing benchmarks
- 12% gap between best AI and human; 33.3% gap between average AI and human
