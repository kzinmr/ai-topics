# DeepSeek-V3 Technical Report

## Summary

DeepSeek-V3 is a state-of-the-art Mixture-of-Experts (MoE) language model with **671B total parameters** and **37B activated per token**. Trained on 14.8 trillion tokens for $5.576M (2.788M H800 GPU hours). Achieves performance comparable to GPT-4o and Claude-3.5-Sonnet while being fully open-weight.

## Key Innovations

1. **Multi-head Latent Attention (MLA)** — Low-rank KV cache compression for efficient inference
2. **DeepSeekMoE** — High-efficiency MoE with fine-grained expert segmentation and shared expert isolation
3. **Auxiliary-Loss-Free Load Balancing** — Dynamic bias-based expert routing without performance-degrading auxiliary loss
4. **Multi-Token Prediction (MTP)** — Predicts multiple future tokens per position; enables 1.8x speculative decoding speedup
5. **FP8 Mixed Precision Training** — First validation of FP8 at 671B scale with fine-grained quantization
6. **DualPipe Algorithm** — Pipeline parallelism with near-zero communication overhead

## Training Details

- **Pre-training tokens**: 14.8T
- **Context extension**: 4K → 32K → 128K (via YaRN)
- **Total GPU hours**: 2,788K H800 hours
- **Estimated cost**: $5.576M
- **Stability**: Zero irrecoverable loss spikes, no rollbacks
- **Post-training**: SFT + GRPO (Group Relative Policy Optimization) + distillation from DeepSeek-R1

## Infrastructure & Systems

- **FP8 mixed precision**: Tile-wise (1×128) activation quantization, block-wise (128×128) weight quantization
- **DualPipe**: Overlaps forward/backward computation with all-to-all communication
- **Cross-node all-to-all**: Overlaps with computation, hidden behind RMSNorm/MLA recomputation
- **Memory optimization**: EMA parameters in CPU, recomputed activations

## Hardware Recommendations (from paper)

1. Communication co-processor to offload all-to-all from SMs
2. Higher-precision FP8 accumulation in Tensor Cores
3. Native support for fine-grained quantization with scaling factors
4. Online quantization fused with memory access (TMA)

## Historical Significance

DeepSeek-V3 demonstrated that open-source models can achieve frontier performance at a fraction of the cost of Western competitors. The $5.576M training budget, FP8 at scale, zero-rollback stability, and novel architectural innovations (MLA, MTP, auxiliary-loss-free load balancing) collectively represent a milestone in efficient LLM training.
