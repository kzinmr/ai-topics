# Kore.ai Launches Artemis Agent Platform on Microsoft Azure

**Source:** [IT Brief US](https://itbrief.news/story/kore-ai-launches-artemis-ai-platform-on-microsoft-azure-4bc9b2ee-2660-4f61-a093-96fec3675af3)
**Date:** May 26, 2026

## Overview

Kore.ai launched Artemis edition of its Agent Platform on Microsoft Azure. Enterprise multi-agent platform with governance embedded at the architectural level.

## Three Core Components

### 1. Agent Blueprint Language (ABL)
Compiled declarative language for defining, validating, governing AI agents. Six orchestration patterns: Supervisor, Delegation, Handoff, Fan-out, Escalation, Agent-to-agent federation.

### 2. Arch (AI Agent Architect)
Converts business objectives into production-ready ABL. Full agent lifecycle, topology design, refinement from production traces.

### 3. Dual-Brain Architecture
Two cognitive engines in parallel via shared memory under single runtime. Combines agentic reasoning with deterministic workflows for predictability and auditability.

## Azure Integrations
Microsoft Foundry, Microsoft Agent 365, Entra ID, Microsoft Graph API, Azure Bot Framework (Teams). Broader cloud later.

## Governance & Compliance
SOC 2 Type II, ISO 27001, PCI DSS, FedRAMP Moderate, HIPAA, HiTrust, GDPR. Real-time PII tokenization, tenant isolation, immutable audit trails.
