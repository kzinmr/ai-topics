# GenAI Handbook

> **Author:** William Brown ([@willccbb](https://x.com/willccbb))
> **Version:** v0.1 (June 5, 2024)
> **Site:** https://genai-handbook.github.io
> **Source:** https://github.com/genai-handbook/genai-handbook.github.io

A structured, textbook-style roadmap organizing the best scattered web resources (blogs, videos, papers) for mastering modern Generative AI. The handbook has 9 sections covering from foundational ML to multimodal systems.

## Structure

- **Section I: Foundations of Sequential Prediction** — ML basics, time-series, RL, Markov models
- **Section II: Neural Sequential Prediction** — RNNs, LSTMs, Transformers, decoder-only architecture
- **Section III: Foundations for Modern Language Modeling** — Tokenization, RoPE, MoE, scaling laws, FSDP
- **Section IV: Finetuning Methods for LLMs** — Instruction tuning, LoRA, RLHF, DPO, context scaling, distillation, merging
- **Section V: LLM Evaluations and Applications** — Benchmarks, sampling, prompting, RAG, agents, DSPy, synthetic data, mechanistic interpretability
- **Section VI: Performance Optimizations for Efficient Inference** — Quantization, speculative decoding, FlashAttention, KV caching, vLLM, CPU offloading
- **Section VII: Sub-Quadratic Context Scaling** — Sliding window, Ring Attention, RWKV, SSMs/Mamba, HyperAttention
- **Section VIII: Generative Modeling Beyond Sequences** — VAEs, GANs, conditional GANs, diffusion
- **Section IX: Multimodal Models** — VQ-VAE, Vision Transformers, multimodal architectures

## Key Features

- Links to quality explainers (blogs, videos, papers) rather than being a textbook itself
- Multiple formats per topic (video + blog + paper)
- Git-style "speedrun tips" (e.g., Karpathy's "Let's build GPT" for Transformers in 2 hours)
- Organizes content from prerequisites → advanced topics
- Explicitly marks topics that can be skipped depending on learning goals
