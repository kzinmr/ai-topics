# Anthropic Orbit Leaked

On May 5, 2026, Anthropic's new proactive assistant called "Orbit" was leaked inside Claude Cowork.

## What Orbit Does
- Auto-generates briefings and insights from Gmail, Slack, GitHub, Calendar, Drive, and Figma
- No prompting required — proactively surfaces personalized insights
- Users can deploy and pin "Orbit apps" for quick access
- Seen as Anthropic's answer to similar proactive assistant features from competitors

## Relationship to Claude Code Leak (March 31, 2026)
The earlier Claude Code source leak revealed several unreleased feature flags:
- **KAIROS**: Always-on daemon running 24/7, acts on user's behalf while idle
- **DREAM**: Self-maintaining memory consolidation system that reorganizes agent knowledge nightly
- **ULTRAPLAN**: Offloads deep planning to remote Opus 4.6 for up to 30 minutes
- **COORDINATOR_MODE**: Multi-agent swarm with structured research-synthesis-implementation phases

Orbit appears to be the productized version of these dormant capabilities, surfacing them in the Claude Cowork interface.
