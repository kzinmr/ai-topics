# OpenAI Is Cooking, The Anthropic Sweep, and SpaceX Courts Cursor

**Author:** Alex Banks | **Date:** Apr 26, 2026

## AI Highlights

### 1. OpenAI Is Cooking (Four Launches in One Week)

- **ChatGPT Images 2** — Next-gen image model: 2K resolution, legible text across non-Latin scripts, Thinking mode with web search mid-generation. Hit #1 on all Image Arena leaderboards; beat Google's Nano Banana models by +242 points (largest gap ever) in Text-to-Image.
- **GPT-5.5** — First fully retrained base model since GPT-4.5. Designed for multi-step work (planning, tool use, self-checking). Scored 82.7% on Terminal-Bench 2.0. Key unlock for agent deployment: can carry tasks through to completion.
- **Workspace Agents** — Codex-powered shared agents for Business/Enterprise plans, integrating Slack, Salesforce, Notion, Google Drive with persistent memory and role-based governance.
- **ChatGPT for Clinicians** — Free tool for verified US physicians, NPs, PAs, pharmacists; includes cited medical sources and HIPAA options.

### 2. The Anthropic Sweep (Reach & Memory)

- **Live artifacts in Cowork** — Dashboards/trackers stay connected to apps/files, pulling fresh data on reopen.
- **Memory for Managed Agents** — Agents on Claude Platform can now learn across sessions. Memory stored as editable text files (exportable, auditable, versionable, rollback via API) instead of a black-box vector store.
- **Consumer connectors** — Claude now plugs into 15 new everyday apps: Booking.com, Resy, Spotify, Audible, Instacart, AllTrails, Thumbtack, TurboTax, Uber, etc. Directory now past 200 connectors.

### 3. SpaceX Courts Cursor (Deep Partnership)

- **Deal structure:** SpaceX has the right to acquire Cursor later in 2026 for $60 billion, or pay Cursor $10 billion for collaborative work (likely compute credits — a subsidy with an exit attached).
- **Why Cursor needed it:** Cursor's own model Composer 2 was built on Moonshot's Kimi; community response was lukewarm. A million H100 equivalents via SpaceX's Colossus gives them a real shot at training a frontier model.
- **The AI lab meta (via Kevin Kwok):** Top coding labs now need to own both model and product. Distribution without a model is a rental agreement; every dev tool company becomes either a model company or a feature of one.

### Altman's Antidote (World ID 4.0)

- **World ID 4.0** — "full-stack proof of human" infrastructure using iris scanning (the Orb).
- **Integrations:** Tinder (verified-human badges in US), Zoom ("Deep Face" cross-checks iris-scanned image, live selfie, video frame), DocuSign (proof-of-human to signatures), Shopify, Okta, AWS, Vercel, Visa building on top.
- **AgentKit** — lets AI agents carry cryptographic proof they act for a verified human. Vercel has "human in the loop" live; Okta planning "Human Principal" for API policies.
- **Stats:** 18 million people across 160 countries already scanned irises at an Orb.

### Mythos Breach

- Anthropic's "too dangerous to release" model Mythos was accessed on day one by four people in a private Discord. They guessed the endpoint URL (from naming conventions + Mercor breach leak) and used a contractor's legitimate evaluation credentials.
- They reportedly used the model to build simple websites.
