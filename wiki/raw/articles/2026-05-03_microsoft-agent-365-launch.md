# Microsoft Agent 365 — Enterprise AI Agent Governance Control Plane

**Source:** AllInOneAICenter, May 1, 2026
**URL:** https://allinoneaicenter.com/blog/new-ai-tools-may-2026

## Overview

On May 1, 2026, Microsoft launched **Agent 365** — a dedicated governance and security control plane for enterprise AI agents. Priced at $15 per user per month, it is a separate product from Microsoft 365 Copilot. It manages agents built on Microsoft AI platforms (Foundry, Copilot Studio) and third-party agents, giving enterprise IT teams visibility and control over autonomous AI systems running across their organization.

## Key Distinction from Wave 3

Wave 3 (March 9, 2026) brought AI into Office apps. Agent 365 is the **security and governance layer** that sits above those apps, addressing the oversight gap for enterprises deploying autonomous agents.

## Target Audience & Pricing

- **Who:** Enterprise IT and security teams
- **Pricing:** $15/user/month, separate from existing Microsoft 365 subscriptions
- Not relevant for individual users or small businesses

## Competitive Context

This is Microsoft's direct competitor to Google's Gemini Enterprise Agent Platform (announced at Cloud Next 2026, April 23) and Amazon's Bedrock AgentCore. The "how do we govern it?" question has become the new bottleneck in enterprise AI agent adoption.
