# Coder Launches Self-Hosted AI Coding Agents for Enterprise

Coder announced the beta release of **Coder Agents** — a native AI coding agent designed to run entirely on self-hosted infrastructure with model-agnostic support.

## Key Details

- 61% of engineering teams already running AI agents
- 70% of companies deploy agents on infrastructure never designed to support them
- Coder Agents run entirely on customer-owned infrastructure (control plane, orchestration, execution)
- Supports fully air-gapped or network-restricted environments
- Model-agnostic: Anthropic, OpenAI, Google, AWS Bedrock, and self-hosted models
- Beta available now with full feature access and no usage-based limits through September 2026

## Developer & Platform Team Benefits

- Developers: delegate tasks (code writing, test generation, repo analysis, PR creation) via conversational interface and API
- Platform teams: standardized deployment, centralized model access, policy enforcement, visibility into agent usage

CEO Rob Whiteley: "Companies are being forced to choose between adopting AI agents and maintaining control over their infrastructure and data. Coder Agents removes that tradeoff."
