# AWS Agent Toolkit for AWS

**Source:** AWS What's New  
**URL:** https://aws.amazon.com/about-aws/whats-new/2026/05/agent-toolkit/  
**Date:** May 6, 2026

## Summary
AWS launched the Agent Toolkit for AWS, a production-ready suite helping AI coding agents build on AWS with fewer errors, lower token costs, and enterprise-grade security. Successor to AWS Labs MCP servers.

## Key Points
- 40+ agent skills across IAC, storage, analytics, serverless, containers, AI services
- AWS MCP Server (GA): IAM guardrails, CloudWatch/CloudTrail observability, sandboxed execution
- Three plugins: AWS Core, AWS Data Analytics, AWS Agents
- Available at no additional charge; pay only for AWS resources used
- Available in us-east-1 and eu-central-1
- Each skill rigorously evaluated for accuracy and reliability
