# Tracking MCP Server Growth

Source: https://martinalderson.com/posts/tracking-mcp-server-growth/?utm_source=rss&utm_medium=rss&utm_campaign=feed

I've been curious about how fast the MCP ecosystem is actually growing. There's a lot of buzz around it, but I wanted to see real numbers. So I built a tracker that monitors MCP servers from the
official MCP server registry
.
You can check it out here:
mcp-tracker.martinalderson.com
The data shows steady growth. In just the past week, we've gone from around 900 servers to over 1,150. That's roughly 25-30 new servers being published every day.
If you're building MCP servers or using them in your workflow, I'd be curious to hear what you're working on -
drop me a note
. The tracker updates daily, so you can watch the ecosystem evolve in real time.
This is getting interesting.
