# Anthropic blocks OpenClaw from Claude subscriptions in cost crackdown

The Next Web (Apr 2026)

## Key Facts

- **Effective date:** April 4, 2026 — third-party tools blocked from Claude Pro/Max subscriptions
- **Rationale:** Subscriptions weren't built for autonomous agent usage patterns; "outsized strain on systems"
- **Scale:** 135,000+ OpenClaw instances running at time of ban
- **Compute gap:** Single OpenClaw instance running full day = ~300x tokens of typical chat user
- **Cost impact:** Users face API rates of $3/million input tokens or pre-purchase bundles (up to 30% discount)

## Key Quotes

**Boris Cherny (Head of Claude Code, Anthropic):**
> *"Our subscriptions weren't built for the usage patterns of these third-party tools. Capacity is a resource we manage thoughtfully and we are prioritizing our customers using our products and API."*

**Peter Steinberger (OpenClaw creator):**
> *"First they copy some popular features into their closed harness, then they lock out open source."*

**Steinberger on negotiations:**
> *"We told Anthropic that we have many users who only signed up for their sub because of OpenClaw and that it'd be a loss if they cut them off. Now they try to bury the news on a Friday."*

## Concessions

- One-time credit equal to monthly subscription cost (redeemable by April 17, valid 90 days)
- Pre-purchase extra usage bundles at up to 30% discount
- Separate API key option (full API rates, bypasses subscription limits)
