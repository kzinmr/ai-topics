# Prime Intellect: verifiers v1 Announcement

> **Tweet by @PrimeIntellect**, July 12, 2026

Today, we are releasing verifiers v1 — an overhaul of our environment stack for the modern era of agentic RL and evals.

We decompose environments into a taskset, a harness, and a runtime.

Run complex agentic tasks like coding and computer use at scale, in any harness.

## Key Concepts

- **Taskset**: The definition of what the agent must accomplish
- **Harness**: The framework that orchestrates agent interaction with the environment
- **Runtime**: The execution context where agent actions take place

This decomposition represents a modular architecture for RL environments, enabling users to mix and match components across different agentic tasks.
