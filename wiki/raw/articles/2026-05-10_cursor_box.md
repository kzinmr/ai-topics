The largest companies in the world trust Box with their most sensitive content and data. Last year, Box looked to partner with an enterprise-ready agentic coding platform that could accelerate roadmap throughput without compromising product security and quality.
Box selected Cursor for its native controls on data privacy and agent quality. Over 85% of developers at Box now use Cursor daily, driving a 30-50% increase in product roadmap throughput. Using Cursor, Box is completing major migrations 80-90% faster while improving overall product quality and security.
#
Selecting an agentic coding platform built for enterprise scale
Customers trust Box because of its emphasis on security, privacy, and governance controls. When Box was evaluating AI coding tools, it selected a partner that shared the same ethos and execution. "We vet developer tools very seriously. Cursor stood out from other vendors for its data privacy and security controls. Their engineering and security teams worked closely with us to make it an amazing partnership," explained Swaroop Butala, a senior director of engineering.
It became obvious that Cursor is one of the clear leaders to bet on. Cursor is now rolled out throughout the organization as our core IDE and agent platform.
Aaron Levie
CEO, Box
#
Box sets agent guardrails and standards in Cursor
With Cursor, Box's developers define agent guardrails directly in code, eliminating the drift that comes from relying on sprawling documentation or informal conventions. Mick Ryan, a staff engineer, built a standard "AI toolkit" for frontend development using custom Cursor rules.
Another team used Cursor to increase automated test coverage. "We've unlocked many use cases by extending Cursor with rules and commands. It's been an incredible experience building with Cursor while keeping customer safety and security at the forefront," said Ryan.
Cursor is actually bringing better products to our customers. We're building better enterprise-grade controls, better security, and better accessibility.
Swaroop Butala
Senior Director of Engineering, Box
#
30-50% improvement in roadmap delivery and 80-90% faster migrations
Box measures the value of AI coding tools against one main metric: Is delivery throughput on the product roadmap increasing? Since adopting Cursor, Levie estimates that roadmap delivery has increased by 30-50%: "We're doing current work much faster and taking on more as a result."
Cursor allows us to take on bigger chunks of work than we could have before. This is strategic because we can deliver more value for customers at better quality, security, and control.
Aaron Levie
CEO, Box
A long-planned React migration to unify cross-platform feature deployment had stalled in its earliest stages. With Cursor, Box built a proof of concept against the production codebase in three days and completed the overall migration 80% faster than expected. "We created rules so Cursor could understand exactly how the new components could be structured, making sure the migration is safe, secure, and well-executed," said Ryan.
Box applied a similar approach to complete a large design system migration 90% faster than expected. "Before Cursor, this migration would have taken a developer a full quarter. We are now wrapping up the project in about a week," said Butala.
Box prides itself on building enterprise-grade products with a focus on security and quality. Cursor lets us turbocharge these values so we can move faster without compromise.
Swaroop Butala
Senior Director of Engineering, Box
#
Build communities around Cursor to scale adoption
Adoption has also scaled quickly. Over 800 developers now use Cursor daily, with 85% adoption across Box's engineering organization. Cursor made an immediate impact on every developer, from developers already writing a majority of code with AI to those that were brand new to agents. "Cursor has enabled a mindset shift where developers don't feel limited by their prior knowledge," explained Butala.
To support adoption, Box created a formal mentorship program that paired Cursor power users with newer adopters. Mentors spent one hour a week walking mentees through real-world tasks they had solved with Cursor. This program led to a 75% increase in Cursor usage and a 25% increase in the number of Cursor power users across Box in just six weeks.
With Cursor embedded across the SDLC, developers have been freed up from drudgery like codebase maintenance and can focus instead on building value for customers. Ryan put it simply: "The world has so many problems to be solved with Cursor. Now is the time to build."
