# Vellum AI: Agentic Workflows in 2026 — 3 Levels, 4 Components

**Source:** https://www.vellum.ai/blog/agentic-workflows-emerging-architectures-and-design-patterns
**Published:** December 4, 2025
**Contributors:** Yohei Nakajima, Zac Haris, Eduardo Ordax, Armand Ruiz, Erik Wikander, Vasilije Markovic, Anton Eremin

## Key Concepts

### 3 Levels of Agentic Architectures
1. **AI Workflow**: Makes output decisions based on prompts (Level 1)
2. **Router Workflow**: Chooses tasks/tools in predefined environment (Level 2) — where most innovation occurs
3. **Autonomous Agent**: Creates new tasks/tools, writes code, seeks feedback (Level 3) — not yet production-ready

### 4 Core Components
1. **Planning**: CoT, ReAct, Self-Refine, RAISE, Reflexion, LATS, PlaG strategies
2. **Execution**: Tools, subagents, self-tool-creation (LATM), guardrails
3. **Refinement**: LLM-based evaluation, memory systems (vector/KV/graph), human-in-the-loop
4. **Interface**: Human-Agent interface, Agent-Computer Interface (ACI)

### 2026 AI Agent Stack
- Tracing & Replay, LLM Calls with Fallbacks, Human Approval in Production
- Tool Library & Execution, Executable Code
- Metrics & Evaluation, User Feedback Integration, Version Control for Prompts/Models
