# PowerShell Frontier Transformation with Copilot and Agents — Wave 3

**Source:** Microsoft 365 Blog  
**Date:** March 9, 2026  
**URL:** https://www.microsoft.com/en-us/microsoft-365/blog/2026/03/09/powering-frontier-transformation-with-copilot-and-agents/

## Summary

Microsoft announced Wave 3 of Microsoft 365 Copilot, shifting from AI assistance to embedded agentic capabilities built in collaboration with Anthropic. Key products: Copilot Cowork (long-running autonomous agent), Agent 365 (governance control plane), and Microsoft 365 E7 Frontier Suite ($99/user/month).

## Key Products

### Copilot Cowork
- Built with Anthropic, using technology powering Claude Cowork
- Multi-model: combines OpenAI and Anthropic models automatically per task
- Work IQ: full context of employee's work (files, meetings, chats, calendar)
- Long-running, multi-step execution — works autonomously for minutes/hours
- Observable and steerable by user

### Agent 365
- Centralized management layer for IT to observe, secure, and govern every agent
- Integrates Microsoft Admin Center, Defender, Entra, Purview
- $15 per user per month
- Generally available May 1, 2026

### Microsoft 365 E7 Frontier Suite
- Unifies productivity, AI, and security
- Includes: M365 Copilot + Agent 365 + Entra Suite + E5
- $99 per user per month (retail)
- Available May 1, 2026

### Agents in Core Apps
- Word & Excel agents generally available
- PowerPoint agent rolling out (understands layouts, brand kits)
- Outlook agent rolling out (drafts/refines emails)
- Copilot Chat supports MCP Apps (Adobe, Monday.com, Figma)

## Strategic Context
- Microsoft spending ~$500M annually on Anthropic models
- Claude available in Copilot Chat via Frontier program
- IDC projects 1.3 billion AI agents by 2028
- Shift from "single-shot" AI assistance to agentic workflow execution
