## Episode Summary
Wes and Scott talk with Armin Ronacher (Flask author) and Mario Zechner (libGDX author, pi creator) about PI, a minimalist agent harness powering tools like OpenClaw. They unpack why Bash is "all you need," the risks of agents, workflow adaptability, and where AI coding agents are actually headed.

## Key Quotes & Insights
- Mario's design philosophy: *"If I don't need it, it won't be built."*
- Armin and Mario both advocate minimal toolsets (bash + read/write/edit) over bloated agent feature sets
- Discussion of security risks in coding agents and prompt injection
- Practical advice on memory/search integration for agents
- Pi's embeddable SDK architecture enables second-party apps (OpenClaw, MoltBot)
