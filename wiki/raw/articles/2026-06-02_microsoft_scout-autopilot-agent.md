# Quoted Content from Microsoft 365 Blog

Microsoft introduced a new category of agents called Autopilots — always-on agents that work autonomously, with their own identity, and act on your behalf. Microsoft Scout is the first Autopilot agent.

Key features:
- Always-on background operation, acts without needing to be prompted each time
- Operates with its own identity (Entra ID governed), with permissions and policies
- Integrated across M365: Teams, Outlook, OneDrive, SharePoint
- Connects to chats, email, calendar, documents
- Proactively schedules meetings across time zones, flags important meetings, generates prep materials
- Work IQ: learns how you work, what you care about, what needs to happen next
- Built on OpenClaw open-source technology
- Contributes policy conformance upstream to OpenClaw
- Enterprise-grade: Entra identity per agent, credentials scoped to task, redacted from logs
- Sensitive actions can require human sign-off
- Microsoft Purview data protection integration
- Private preview through Frontier program
- Requires GitHub Copilot license, Intune policy configuration, Frontier enrollment
- Early desktop experience used internally by Microsoft employees
