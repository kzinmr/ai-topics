# Claude for Creative Work — Anthropic Creative Coalition

**Release Date:** April 28, 2026
**Core Mission:** To integrate Claude into the professional creative workflow by handling repetitive tasks, automating manual toil, and expanding the technical capabilities of artists and designers.

## New Creative Tool Connectors
Anthropic released 9 connectors letting Claude interact directly with industry-standard software:

| Tool | Functionality |
|------|--------------|
| Adobe for Creativity | Accesses 50+ tools across Creative Cloud (Photoshop, Premiere, Express) |
| Autodesk Fusion | Enables natural language conversation to create/modify 3D models |
| Blender | Natural-language interface to Python API; MCP-based connector |
| Affinity by Canva | Automates production tasks: batch image adjustments, layer renaming, file exports |
| SketchUp | Converts descriptions of rooms/furniture into 3D model starting points |
| Ableton | Grounds Claude in official product documentation for Live and Push |
| Splice | Royalty-free sample catalog search directly within Claude |
| Resolume (Arena/Wire) | Real-time natural language control for VJs and live visual artists |

## Key Capabilities
- On-Demand Tutoring: Explaining complex software concepts
- Extending Tools with Code: Claude Code writes scripts, plugins, generative systems
- Pipeline Bridging: Translating formats across design, 3D, audio applications
- Rapid Exploration: Claude Design for visualizing software experiences
- Production Automation: Project scaffolding and batch-processing assets

## Open Source Commitment
- Blender connector built on MCP (Model Context Protocol) — interoperable with other LLMs
- Anthropic made a donation to support Blender's Python API development

## Educational Partnerships
- RISD: Art and Computation program
- Ringling College of Art and Design: Fundamentals of AI for Creatives
- Goldsmiths, University of London: MA/MFA Computational Arts

> "Claude can't replace taste or imagination, but it can open up new ways of working."
