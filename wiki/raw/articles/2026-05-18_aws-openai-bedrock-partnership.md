# AWS and OpenAI Expanded Partnership: Frontier AI on Amazon Bedrock

**Source:** About Amazon News  
**URL:** https://www.aboutamazon.com/news/aws/bedrock-openai-models  
**Date:** May 2026

## Summary
AWS announced a major expansion of its partnership with OpenAI, bringing three new offerings (all in limited preview): OpenAI models on Amazon Bedrock, Codex on Amazon Bedrock, and Amazon Bedrock Managed Agents powered by OpenAI.

## Key Points
- Latest OpenAI models available through Bedrock's unified API and controls
- Codex on Amazon Bedrock brings AI-powered software development into AWS environments
- Bedrock Managed Agents combines OpenAI frontier reasoning with AWS's infrastructure, memory, skills, identity, and compute
- Enterprise-grade security: IAM, PrivateLink, encryption, CloudTrail logging
- Financial governance: OpenAI usage counts toward existing AWS cloud commitments
- Bedrock Managed Agents built with OpenAI Agent Harness, optimized for the models
- AgentCore serves as the default compute environment
- Box CTO Ben Kus endorsed the partnership for enterprise agent deployment
