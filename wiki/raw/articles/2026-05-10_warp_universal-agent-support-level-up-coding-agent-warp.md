# Introducing Universal Agent Support: level up any coding agent with Warp

Introducing Universal Agent Support: level up any coding agent with Warp
Zach Lloyd
April 14, 2026
Today we are excited to announce a host of new features that make Warp the best terminal for multi-threaded development with the most popular coding agent CLIs, including Claude Code, Codex, Gemini CLI, and OpenCode.
Our belief is that the terminal is already the best place to do agentic development, but traditional terminals are lacking some key features for working with agents.
So today, we are making coding agents easier to manage with vertical tabs, notifications, native code review and more. These are the same features that make our native agent powerful, but for any coding agent you might prefer to use.
Our goal here is to build the best workbench for shipping with coding agents at scale, and supporting the entire ecosystem of agents furthers that goal. Stay tuned for more from us in this direction.
What's in the box
Vertical tabs
You can now group agents into vertical tabs and configure metadata like git branch, worktrees, and pull-requests at a glance. Our vertical tabs implementation is super customizable to support whatever workflow you prefer.
Tab Configs
We now support Tab Configs — a revamped, tab-level schema that lets you create your ideal tab setup including directory, startup commands, theme, worktree management with a single click.
Notifications
Warp pipes through notifications from any CLI coding agent, both in-app and via system notifications, so you know exactly when your attention is needed.
A unified notification center that works across all coding agents and harnesses, giving you a single place to track what needs your attention.
Code review
Send inline comments from Warp's code review directly to a running third-party agent session.
Attach code as context
Select code, files, or snippets and feed them to your running agent without copy-pasting or switching tools.
Rich input
Use Warp's rich terminal input to enter complex multiline prompts in any coding agent. Mouse control, cut / copy / paste, voice input — it all just works. Access saved /prompts, /skills in scope, and @context directly from the editor.
Attach images as context, open your file explorer, and view your code changes through the agent toolbar.
Remote control
Publish any CLI coding agent session to the cloud so you (and your teammates) can access it on the go. Monitor progress and steer any coding agent from your phone or another computer.
