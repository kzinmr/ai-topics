# Anthropic Publishes 2-Hour Training on Building Claude Agents

> Anthropic a publié une Formation complète de 2 HEURES sur la construction d'agents Claude.
>
> Animé par l'ingénieur qui construit Claude Code.
>
> De A à Z : Structurer un agent qui se gère sans supervision. Lui donner accès au terminal pour exécuter des commandes, manipuler des fichiers, et itérer sur son propre travail.

## Context

Tweet by @Jouhatsu_ai (May 16, 2026) reporting that Anthropic has published a comprehensive 2-hour training course on building Claude agents. The course is led by the engineer who builds Claude Code, covering agent structure, terminal access, file manipulation, and autonomous iteration.

## Related Anthropic Training Resources

Anthropic offers multiple training pathways on their Skilljar platform (anthropiccertifications.com):

- **Claude Code in Action** (course 6, Intermediate): Integrate Claude Code into real development workflows — tool use architecture, context management, custom automations, MCP server integration, GitHub workflows, thinking/planning modes
- **Introduction to Agent Skills** (course 7, Intermediate): Build, configure, and distribute reusable Skills for Claude Code — SKILL.md frontmatter, descriptions, directory organization, enterprise deployment
- **Introduction to Subagents** (course 8, Intermediate): Delegate tasks to isolated sub-agents in Claude Code — separate context windows, structured outputs, obstacle reporting
- **Building with the Claude API** (course 2, Intermediate): Full Anthropic API — tool use, RAG, prompt caching, streaming, extended thinking, agentic patterns

These courses prepare for the Claude Solutions Architect certification exam.

Additionally, a 7-hour free video course (completeaitraining.com) covers "From CLI to Agent SDK" — setup, context, hooks, custom commands, GitHub reviews, and Agent SDK.
