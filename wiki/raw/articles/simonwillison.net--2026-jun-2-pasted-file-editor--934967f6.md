# Pasted File Editor

Source: https://simonwillison.net/2026/Jun/2/pasted-file-editor/#atom-everything

I really like how you can paste a large volume of text into
claude.ai
(or the Claude desktop/mobile apps) and it will detect it as a large paste and turn it into a file attachment instead.
I decided to have Codex desktop
build me a version of that
as a prototype.
You can also open files directly - including images which will be shown as thumbnails - or drag files onto the textarea.
