# Google Antigravity 2.0

Google's AI coding assistant, available as:
- **Antigravity CLI**: Work directly in your codebase. Build, debug, ship from terminal.
- **Antigravity IDE**: Standalone desktop IDE (macOS, Windows, Linux).
- **Antigravity SDK**: Programmatic access.

Install CLI: `curl -fsSL https://antigravity.google/cli/install.sh | bash`

Platforms: macOS (Apple Silicon/Intel), Windows (x64/ARM64), Linux (x64/ARM64).
Requires authentication with Antigravity or Antigravity IDE before CLI use.
