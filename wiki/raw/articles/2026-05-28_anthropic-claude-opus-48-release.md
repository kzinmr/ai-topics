# Introducing Claude Opus 4.8

Anthropic upgraded Claude Opus to version 4.8, building on Opus 4.7 with improvements across benchmarks.

Key features:
- More effective collaborator: 4x less likely to let code flaws pass unremarked
- Flags uncertainty in its own outputs
- Pushes back on weak plans
- "Adaptive thinking" replaced with "Effort" control: Low/Medium/High/Extra/Max
- Separate Thinking toggle available
- Fast mode for Opus 4.8: 2.5× speed, 3× cheaper than previous models
- Available at same price as Opus 4.7
- Testers report sharper judgment in agentic tasks
