# Claude Managed Agents: Self-Hosted Sandboxes (Public Beta) + MCP Tunnels (Research Preview)

Announced live at Code with Claude London on 2026-05-19:

> We're launching self-hosted sandboxes (public beta) and MCP tunnels (research preview) in Claude Managed Agents. Run agents inside your own perimeter, with your security controls applied by default.

## Key Points

- **Self-hosted sandboxes** — Public beta. Agents run inside the user's own infrastructure perimeter, with the user's security controls applied by default. This is a significant move for enterprise adoption where security and data residency requirements prevent using managed cloud sandboxes.
- **MCP Tunnels** — Research preview. Allows Claude Managed Agents to connect to local MCP servers behind firewalls/NAT via secure tunnels, expanding the range of tools agents can access in enterprise environments.

Announced alongside a video demo at the Code with Claude London event.
