# GPT-5.5 Instant

OpenAI updated ChatGPT's default model (May 5, 2026) to GPT-5.5 Instant with significant improvements:

- 52.5% fewer hallucinated claims vs GPT-5.3 Instant on high-stakes prompts (medicine, law, finance)
- 37.3% reduction in inaccurate claims on user-flagged challenging conversations
- Improved visual reasoning (analyzing photo/image uploads)
- Better STEM question answering
- Smarter decisions about when to use web search
- More concise responses without losing substance
- Better personalization based on user context
- Self-correction: recovers from initial mistakes by rechecking work (notable improvement shown in math example)
