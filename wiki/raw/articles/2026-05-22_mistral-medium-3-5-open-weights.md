# Mistral Medium 3.5 announcement

Mistral AI released Mistral Medium 3.5, a 128B dense model (not MoE) with 256k context window, open weights under modified MIT license.

Key features:
- Merges instruction-following, reasoning, and coding into single model
- Configurable reasoning effort per request
- Custom vision encoder trained from scratch
- Self-hostable on as few as 4 GPUs
- SWE-Bench Verified: 77.6%
- τ³-Telecom: 91.4

Launched alongside:
- Mistral Vibe remote agents (async cloud coding sessions)
- Work mode in Le Chat (Preview) - multi-step agentic workflows
- Replaces Devstral 2 in Vibe CLI
