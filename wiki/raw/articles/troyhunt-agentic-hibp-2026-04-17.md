# Here's What Agentic AI Can Do With Have I Been Pwned's APIs

- **Source:** Troy Hunt (troyhunt.com)
- **Date:** 2026-04-17
- **URL:** https://www.troyhunt.com/heres-what-agentic-ai-can-do-with-have-i-been-pwneds-apis/
- **Tags:** [agentic-ai, mcp, hibp, cybersecurity]

## Summary

Troy Hunt demonstrates agentic AI use cases with Have I Been Pwned (HIBP) APIs using MCP and OpenClaw on Mac Mini via Telegram bot. Key demos:

1. **MCP Integration:** HIBP MCP server at https://haveibeenpwned.com/mcp — enables AI apps to access breach data via Model Context Protocol
2. **Human Use Cases:** Shows how non-technical users ("normies") can query breach data through AI agents
3. **GitHub Copilot Extension:** JSON config to extend Copilot with HIBP MCP endpoint
4. **Personal breach checking:** Agent queries your own data to see what's been exposed

Troy is "on the fence" about MCP's usefulness but acknowledges it was a no-brainer to create. Demonstrates using OpenClaw as the agent interface for HIBP queries.

## Key Points
- MCP allows AI apps to connect to data sources, tools, and workflows
- HIBP now has an MCP server endpoint
- Can be integrated with GitHub Copilot, Claude, ChatGPT
- Non-technical users can use AI agents to query their own breach data
