# OpenAI and AWS Strategic Partnership Expansion

**Date:** April 28, 2026
**Status:** Limited Preview
**Core Objective:** Integrating OpenAI's frontier models and agentic tools directly into AWS ecosystem

## Three Pillars

### 1. OpenAI Models on Amazon Bedrock
- Frontier models including GPT-5.5 available on Bedrock
- Enterprise controls: IAM, AWS PrivateLink, guardrails, encryption, CloudTrail
- Single path from experimentation to production

### 2. Codex on AWS
- Codex powered by models served from Amazon Bedrock
- All customer data processed by Amazon Bedrock
- Usage can be applied toward existing AWS cloud commitments
- Available via Bedrock API → Codex CLI, desktop app, VS Code extension
- Supports refactoring, legacy modernization, document-based work

### 3. Amazon Bedrock Managed Agents (Powered by OpenAI)
- Build agents that maintain context, execute multi-step workflows, use tools
- AWS handles infrastructure, orchestration, governance
- Built-in integration with Amazon security and compliance controls
- Each agent has identity, logs each action, runs in customer's environment

## Strategic Impact
- Microsoft partnership renegotiation enables multi-cloud deployment
- Enterprises can now use OpenAI on trusted AWS infrastructure
- Represents breaking out of the "Azure-shaped box"
