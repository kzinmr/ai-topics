# Active Crawl Research: June 1, 2026

## Probe&Prefill — Tool Use Efficiency

arXiv:2605.09252 — "LLM Agents Already Know When to Call Tools -- Even Without Reasoning"

Tool-augmented LLM agents tend to call tools indiscriminately. The When2Tool benchmark covers 18 environments across three categories of tool necessity: computational scale, knowledge boundaries, and execution reliability. Key finding: tool necessity is linearly decodable from hidden states (AUROC 0.89-0.96), meaning models "know" when tools are needed but fail to act on it. Probe&Prefill reduces tool calls by 48% with only 1.7% accuracy loss. Code at github.com/Trustworthy-ML-Lab/when2tool.

## Mythos — Anthropic's General Intelligence Assistant

Mythos Preview announced alongside Claude Opus 4.8 (May 28, 2026). Anthropic's most advanced assistant — 5/18 Anthropic engineers believe this version of Mythos could substitute for an L4 engineer with harness improvements. General release expected "in coming weeks." Mythos represents Anthropic's push toward general-purpose AI assistants that can handle complex, multi-step tasks autonomously.

## Autoscientist — Adaption's Automated Frontier Training

Sara Hooker (Cohere/Google alum) founded Adaption, which launched AutoScientist — a tool that helps models train themselves. Automates frontier AI training by having agents make incremental improvements. If researchers start to push the frontier forward, could spiral into recursive self-improvement.

## Recursive Superintelligence — Richard Socher's Company

Richard Socher launched Recursive Superintelligence with RSI as an explicit goal: "build truly recursive, self-improving superintelligence at scale — the entire process of ideation, implementation, and validation of research ideas would be automatic." Part of a broader industry trend: Andrej Karpathy's Auto-Research at Anthropic, Adaption's AutoScientist, Disarray's Kaggle-winning self-trained agent (28 medals).
