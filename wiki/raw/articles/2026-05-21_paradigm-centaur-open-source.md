# Centaur: Multiplayer, Self-Hosted, Secure Agent Runtime

## Source
Paradigm — May 21, 2026

## Key Points
- Self-hosted multiplayer AI agent runtime (Apache 2.0)
- Production use at Paradigm and Tempo since January 2026
- Isolated sandbox containers per Slack thread session
- Iron Proxy: agents never see raw API keys
- Tools added once, available organization-wide
- Durable execution surviving restarts and crashes
- Observability via structured JSON logs
