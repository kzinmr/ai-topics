# Coder Sets a New Standard for AI Coding with Self-Hosted, AI Model Agnostic Coder Agents

**AUSTIN, Texas, May 6, 2026** — Coder, the leader in self-hosted AI development infrastructure for the enterprise, announced the beta release of Coder Agents.

## What is Coder Agents?

A native AI coding agent built directly into the Coder platform, designed to run entirely on **self-hosted infrastructure**. Enterprises can use any AI model they want while maintaining full governance over infrastructure, data, and models.

## Key Features

- **Self-hosted**: The entire agent system — control plane, orchestration, and execution — runs on customer-owned infrastructure
- **Model-agnostic**: Supports Anthropic, OpenAI, Google, AWS Bedrock, and self-hosted models
- **Air-gapped ready**: Works in fully air-gapped or network-restricted environments
- **Enterprise governance**: Centralized policies for model access, prompts, and usage across teams
- **API and conversational interface**: Developers can delegate tasks like writing code, generating tests, analyzing repos, and opening PRs

## Enterprise Rationale

Enterprises in regulated industries need to control where source code and model interactions are processed. Many existing AI coding tools require sending code and context to third-party cloud services, creating data residency, compliance, and auditability challenges.

Coder Agents keeps all activity within the organization's own infrastructure — no source code, prompts, or model interactions leave the network perimeter.

## Developer Workflows Supported

- Writing code
- Generating tests
- Analyzing repositories
- Opening pull requests
- Standardized deployment across teams
