# DNS-AID: Decentralized AI Agent Discovery

The Linux Foundation launched DNS-AID (DNS for AI Discovery), an open source project enabling secure, decentralized discovery and communication between AI agents and MCP servers using existing DNS infrastructure.

## Key Details
- Initially developed by Infoblox, now under Linux Foundation governance
- Uses DNS SVCB/HTTPS records as a vendor-neutral directory
- Reference implementation: Python SDK, CLI, MCP server
- Initial supporters: Cloudflare, CSC, Equinix, GoDaddy, Indeed, Infoblox, ISC, WWT
- Repository: github.com/dns-aid

## Significance
Addresses fragmentation in AI agent discovery (hardcoded URLs, centralized registries) by anchoring discovery in DNS — the internet's most ubiquitous infrastructure. Provides decentralization, global scalability, interoperability, and trust bootstrapping via AgentCards and attestations.
