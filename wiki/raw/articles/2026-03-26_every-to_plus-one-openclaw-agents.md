# Plus One: One-click OpenClaw Agents by Every

Every's hosted OpenClaw agents in Slack, pre-loaded with Every tools/skills/workflows.

## Key Insights
- OpenClaw requires significant manual setup + dedicated machine 24/7 (many buy Mac Minis)
- Hard part of AI agents: infrastructure, hosting, integrations, skills, ongoing care
- Plus One: hosted on Every's secure servers, one-click setup
- Connected to Every ecosystem: Cora (email), Spiral (writing), Proof (docs)
- Pre-loaded skills: content digest, daily brief, animate (Remotion), frontend
- Integrates Google Workspace and Notion
- Onboarding 20/week, scaling up
- Each Plus One = separate virtual cloud server (expensive to run)
- Zero-data-retention LLMs

## Features
- Lives in Slack
- Self-improving with memory
- Works autonomously on recurring schedules
- Collaborative (works with other humans and agents)
