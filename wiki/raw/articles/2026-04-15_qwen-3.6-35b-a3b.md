Qwen3.6-35B-A3B is the first open-weight Qwen3.6 model, a sparse MoE model with 35B total parameters and only 3B active per token.

## Specifications
- Total parameters: 35B
- Active parameters: 3B per token
- Architecture: 40 layers, hybrid (Gated DeltaNet → MoE) + (Gated Attention → MoE)
- Experts: 256 total, 8 routed + 1 shared active
- Context: 262,144 tokens native, extensible to 1,010,000
- License: Apache 2.0
- Weights: BF16, available on Hugging Face and ModelScope

## Key Capabilities
- Agentic coding: competitive with much larger dense models (Qwen3.5-27B, Gemma4-31B)
- Multimodal: supports vision encoder for image understanding
- Thinking preservation: retains reasoning context across conversation turns
- SWE-bench Verified: 73.4%
- GPQA Diamond: 86.0%

## Efficiency
- Runs on single RTX 4090 with quantization
- RTX PRO 6000 (96GB VRAM) can run in BF16 without tensor parallelism
- DFlash speculative decoding achieves up to 6x lossless acceleration
- 3.3M+ Hugging Face downloads in first month

## Novelty
Combines agentic coding focus + thinking preservation + multimodal support + large context + Apache 2.0 open-weight release. Optimized for iterative development workflows, not just static prompts.
