# Rabbithole — Infinite Canvas for Learning

Shlok Khemani (@shloked) released **Rabbithole**, an open-source infinite canvas for learning that integrates with AI coding agents.

## Key Features

- **Infinite canvas UI**: Open documents, ask questions from selections, and branch understanding visually
- **MCP server**: Works as an MCP server for Claude Code, Codex, and any agent
- **Local-first**: API keys stay in your browser; can also be cloned and run entirely on your own machine
- **Selection-based Q&A**: Select text on the canvas and ask questions — answers branch out as new documents

## Links

- Web app: https://rabbithole.ing
- GitHub: https://github.com/shlokkhemani/rabbithole

## Architecture

Built as a browser-based infinite canvas with an MCP server integration, allowing coding agents to read from and write to the canvas. The project emphasizes local privacy (keys never leave the browser) and self-hosting capability.
