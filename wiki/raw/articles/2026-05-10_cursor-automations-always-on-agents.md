# Cursor Automations announcement

Cursor introduced Automations - always-on AI agents running on schedules or triggered by events.

Features:
- Triggered by: Slack messages, Linear issues, GitHub PRs, PagerDuty incidents, webhooks, schedules
- Cloud Cloud execution with MCP integration
- Memory tool for learning from past runs
- Self-verifying output

Use cases at Cursor:
- Bugbot: PR review automation (thousands of triggers daily)
- Security review: Push-to-main vulnerability auditing
- Agentic codeowners: Risk classification and auto-approval
- Incident response: PagerDuty integration with Datadog MCP

Enterprise adoption: Rippling uses for personal assistant, Jira creation, incident triage
