# Mistral Medium 3.5 と Vibe リモートエージェント

## 概要
Mistral AI は2026年4月、**Mistral Medium 3.5** と、それを基盤とする**Vibe リモートコーディングエージェント**を発表した。コーディングエージェントをローカル環境からクラウドサンドボックスに移行し、並列・非同期実行を可能にするアーキテクチャである。

## Mistral Medium 3.5 の技術仕様
- **モデル構成**: 128B デンモデル（単一の重みセット）
- **コンテキストウィンドウ**: 256k トークン
- **機能**: 指示追従、推論、コーディングを単一モデルに統合
- **ビジョン**: 可変サイズ・アスペクト比に対応するビジョンエンコーダーをスクラッチから学習
- **推論設定**: リクエストごとに「推論努力」を調整可能（即答〜複雑なエージェント実行）

### ベンチマーク
- **SWE-bench Verified**: 77.6%（Devstral 2、Qwen3.5 397B を上回る）
- **τ³-Telecom**: 91.4

### ライセンスとデプロイ
- 修正 MIT ライセンスでオープンウェイト公開
- Hugging Face で利用可能
- 最小 4 GPU でセルフホスト可能
- NVIDIA NIM として build.nvidia.com でホスト

## Mistral Vibe: リモートコーディングエージェント
Vibe は、コーディングエージェントをローカルラップトップから分離されたクラウドサンドボックスに移行する。

### 主要機能
- **非同期実行**: タスクを開始して離脱可能。完了時に通知を受ける
- **テレポーテーション**: ローカル CLI セッションをクラウドに「テレポート」し、セッション履歴、タスク状態、承認を引き継ぎ
- **並列性**: 開発者がボトルネックになることなく、複数のコーディングセッションを同時実行
- **統合**: GitHub（コード/PR）、Linear/Jira（Issue）、Sentry（インシデント）、Slack/MS Teams（レポート）

### ユースケース
> 「高度なボリュームと明確に定義された作業 — 開発者の時間は取るが判断は取らないもの: モジュールリファクタ、テスト生成、依存関係アップグレード、CI 調査、バグ修正 に適合する」

## Le Chat Work Mode（プレビュー）
Work Mode は Le Chat を実行バックエンドに変え、マルチステッププロジェクトとクロスツールアクションを可能にする。

- **クロスツールワークフロー**: メール、カレンダー、メッセージングのデータを集約（例: 出席者コンテキスト付きの会議準備資料）
- **リサーチと合成**: ウェブと内部ドキュメントを深度調査し、構造化レポートを生成
- **管理自動化**: 受信トレージのトリアージ、返信ドラフト、チャット議論からの Jira Issue 作成
- **透明性と安全性**: すべてのツール呼び出しと「推論根拠」が表示可能。機密アクションには明示的な承認が必要
