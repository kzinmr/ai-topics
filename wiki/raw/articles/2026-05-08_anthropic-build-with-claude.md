# Build with Claude: Comprehensive Development Guide

This guide summarizes the core pillars of building applications with Anthropic's Claude, covering model capabilities, architectural patterns, and developer resources.

## Quick Start & Developer Essentials
- Anthropic Console (API keys and account setup)
- Claude Code: `npm install -g @anthropic-ai/claude-code`
- Quickstart Guide for first API request
- Anthropic Cookbook: code snippets for common workflows
- Anthropic Quickstarts: pre-built application templates

## Advanced Model Capabilities
- **Claude 4.5 Models**: Sonnet and Opus for peak intelligence
- **Extended Thinking**: internal reasoning before responding
- **Vision & Multimodal**: charts, graphs, text extraction from images
- **Computer Use (Beta)**: desktop environment interaction

## Architectural Patterns & Tools
- **Agents & Skills**: autonomous systems with MCP standard
- **Tool Use (Function Calling)**: external APIs and local tools
- **RAG**: external data with Contextual Retrieval, Voyage AI, LlamaIndex, MongoDB

## Optimization & Performance
- **Prompt Engineering**: Prompt Generator, interactive tutorial
- **Prompt Caching**: cost/latency reduction for reused context
- **Evaluations (Evals)**: systematic model testing in Console
