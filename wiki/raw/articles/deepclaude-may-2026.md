# DeepClaude

**Date**: May 4, 2026

**Hacker News**: 567 points, 237 comments

DeepClaude is a new open-source tool that enables using **Claude Code**'s agent loop with **DeepSeek V4 Pro**. It allows Claude to orchestrate DeepSeek models for multi-step tasks.

**Significance**: Demonstrates the emerging pattern of **agent framework interoperability** — where the orchestration layer (Claude Code) is decoupled from the underlying model (DeepSeek V4 Pro). This allows users to leverage Claude's superior agent planning capabilities while using more cost-effective inference from DeepSeek.

**Technical pattern**: Claude Code → orchestration/agent loop → DeepSeek V4 Pro → execution → back to Claude
