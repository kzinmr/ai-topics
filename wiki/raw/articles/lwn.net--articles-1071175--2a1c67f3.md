# NetHack 5.0.0 released

Source: https://lwn.net/Articles/1071175/

NetHack 5.0.0 released
[Posted May 4, 2026 by jzb]
Version 5.0.0
of the
NetHack
dungeon-exploration game, a distant relative of
Rogue
and
Hack
, has been released. NetHack's code is now compliant with the
C99 standard, and the release includes more than 3,100
bug fixes and changes, detailed in
doc/fixes5-0-0.txt
(may contain game spoilers). Saved games from previous versions will
not work with NetHack 5.0.0.
