What's New: June Disclosure 2026
This month, we shipped new data partnerships, Matrix capabilities, and workflow improvements across Chat, Projects, and the platform.
This month, Hebbia ships a significant expansion of data partnerships, Matrix capabilities, and workflow improvements across Chat, Projects, and the platform.
Hebbia API & MCP
Hebbia API
Connect Hebbia directly to any internal system to power real-time insights and automate critical workflows with intelligence from Hebbia.
Hebbia MCP
Ask natural-language questions over your Hebbia projects and data sources in Claude and ChatGPT. Get answers with inline citations without leaving your host environment.
New Data Integrations
Fitch
You can now access data from Fitch, a leading global credit rating agency. Pull reports into Matrix via Browse, or ask Chat to surface ratings and the rationale behind upgrades and downgrades grounded in the source material.
Intercontinental Exchange
You can now answer questions using near-real-time equity and ETF market data, powered by Intercontinental Exchange. Ask about recent price performance, trading volume, volatility, and trends across individual companies or sectors, or chart prices directly in the conversation.
Intralinks
Intralinks, a global leader in AI-enabled dealmaking trusted across M&A, capital markets, and alternative investments, now syncs directly into Hebbia—giving deal teams a continuous workflow from data room to analysis.
Salesforce
Hebbia now connects to Salesforce, bringing CRM data into the same context as your financial and market research. Analyze pipeline status, account coverage, and client activity alongside documents and data sources, all within Chat.
Workflow and Agent Improvements
Project Controls
New controls for managing and navigating shared work across your team. Publish or unpublish items directly from project history, track activity alongside your history in the home tab, and follow breadcrumbs back to the originating project from any open chat or matrix.
Citations in Slides
PowerPoint presentations generated in Chat now include inline citation markers with source references in the slide footer, so every claim in the output is traceable back to its origin.
Tick-and-Tie Agent
A purpose-built agent that verifies figures in a presentation against a source file. Upload your reference data and the document to check, and Hebbia extracts every metric and cross-references it page by page.
Agent Run Controls
Chat agents now prompt you to review and adjust sources before running. Add documents, attach projects, or swap in additional sources at the moment of execution, with the option to disable the step entirely from Agent settings.
Expandable Tables
Tables in Chat can now be expanded or collapsed, making it easier to read dense data and manage screen space within a response.
CSV File Uploads
Upload CSV files directly into Chat alongside your documents. Ask questions about the data, filter and summarize it, or combine it with other sources for cross-referenced analysis.
