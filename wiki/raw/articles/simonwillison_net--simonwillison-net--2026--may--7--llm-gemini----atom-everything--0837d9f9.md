llm-gemini 0.31
May 7th, 2026

Simon released version 0.31 of llm-gemini, his Python CLI plugin for Google's Gemini API via the llm framework.

Key changes:
- Adds support for gemini-3.1-flash-lite (GA release)
- Updates model mappings for the latest Gemini releases
- Bug fixes for authentication and rate limiting

This continues Simon's pattern of quickly releasing CLI tools for new model APIs, making them accessible to developers who prefer command-line workflows over web interfaces.
