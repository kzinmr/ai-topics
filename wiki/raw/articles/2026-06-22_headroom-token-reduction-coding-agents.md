# Headroom — Portable Token-Reduction Skill for Coding Agents

GitHub: https://github.com/roman-ryzenadvanced/headroom-skill
Author: Roman Ryzen Advanced (Netflix engineer)

## Overview

Headroom is a portable token-reduction skill for any coding agent. It wraps local agents (Codex, Cursor, OpenClaw, Hermes, Claude Code, Aider, Goose, Continue, OpenClaw) and reduces token consumption by up to 95% through intelligent context management.

**Reported Results**: 95% token usage reduction for local coding agent workflows.

**Supported Agents**: Claude Code, Codex, Cursor, Aider, Claw, Hermes, Goose, Continue, OpenClaw

## Significance

Token cost is a primary constraint for autonomous agent workflows. A 95% reduction means agents can run 20x longer on the same budget, enabling longer-horizon autonomous tasks without proportional cost increase. This is particularly relevant for:
- Local/hosted agent deployments
- Long-running autonomous coding sessions
- Multi-step agent workflows with large context requirements
