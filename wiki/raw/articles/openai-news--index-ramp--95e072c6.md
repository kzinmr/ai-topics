# How Ramp Engineers Accelerate Code Review with Codex

Source: https://openai.com/index/ramp

## Overview
Ramp's AI Developer Experience team uses Codex with GPT-5.5 to deliver thorough, substantive pull-request feedback in minutes (down from hours), reduce manual effort during code review and on-call incident management, and build internal agentic tools like the On-Call Assistant.

## Key Quotes from Austin Ray (AI DevEx at Ramp)
- "Codex code review catches things that I miss and that other engineers miss and that other AI code reviewers definitely miss."
- "Codex code review is industry gold standard. We've been relying on it for a long time here at Ramp. It's incredible, and our engineers ask for it by name."
- "Codex with GPT-5.5 is incredibly adept at dealing with that complexity in a way that would take me a ton of mental effort, a lot of sleep, and a lot of single-minded focus on the problem to figure out."
- "Our product surface area is pretty immense. Codex with GPT-5.5 handles it like it's nothing."
- "Engineers are going to become orchestrators. The skill is no longer writing every line of code yourself. It's knowing how to direct AI tools like Codex, when to trust them, and when to push back."

## Code Review Impact
- Engineers who previously waited hours for a first review now receive substantive, codebase-aware feedback in minutes.
- Codex reasons deeply against the codebase, providing thoroughness that most human reviewers don't have time for.
- Codex has become a mandatory, trusted part of the code review flow at Ramp.

## On-Call Assistant
Ramp used Codex to build an On-Call Assistant, an agentic tool that offloads much of on-call rotation work. Tackles large business logic, concurrency bugs, and long-running incident investigations. Development speed increased dramatically with GPT-5.5.

## Leadership Recommendations
1. Demonstrate first-hand: guide engineers through a strong initial Codex session
2. Build trust and iteration: transform skepticism into willingness
3. Invest in the feedback loop: maintain direct line to the vendor

## Future Vision
Engineers become orchestrators who direct AI tools, knowing when to trust and when to push back. With Codex freeing up time and mental bandwidth, teams tackle larger challenges.
