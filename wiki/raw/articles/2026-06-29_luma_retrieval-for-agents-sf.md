# Retrieval for Agents: Building retrieval infrastructure for the new and powerful user

**A Hornet evening at AI Engineer World's Fair**

- **Date**: Monday, June 29, 2026
- **Time**: 5:00 PM – 9:00 PM PT
- **Location**: Mission District, San Francisco (short ride from Moscone)
- **Organizer**: Hornet.dev (hosted by Skip Everling)
- **Speaker**: Jo Kristian Bergum, CEO & Co-founder of Hornet
- **Registration**: https://luma.com/ypnd3grq
- **Calendar**: Hornet.dev

## Event Description

Agents do not search like humans. Legacy retrieval infrastructure was tuned for a human typing two words and clicking the top result. Agents are a different and capable user, with different workload characteristics.

Join Hornet during AI Engineer World's Fair for a technical evening on the retrieval engine built for agents.

Jo Kristian Bergum, Hornet's CEO and co-founder, will talk about what the next generation of retrieval systems for agents look like: workload shape, retrieval tool abstractions, relevance across turns, and the unreasonable effectiveness of primitives for agentic retrieval.

We will have special guests on stage, then open the room for Q&A, drinks, food, and technical conversations with the people building and operating agent infrastructure.

If you are serving agent-driven retrieval traffic, building a Search API for agents, or evaluating core infrastructure for high-volume agent workloads, this event is for you.

**Space is limited.**

## What to Expect

- A talk from Jo Kristian Bergum, with special guests on stage
- Open Q&A
- Hosted drinks and food
- A room full of AI engineers, agent builders, and infrastructure-minded founders

## Key Themes

- Workload shape of agent-driven retrieval
- Retrieval tool abstractions for agents
- Relevance across conversational turns
- Primitives for agentic retrieval
- Building retrieval infrastructure for the "new and powerful user" (AI agents)

## Context

Hornet (hornet.dev) is a retrieval engine company. Jo Kristian Bergum was formerly a key figure at Vespa (the open-source big data serving engine), which is widely used for vector search and retrieval-augmented generation (RAG). This event is co-located with the AI Engineer World's Fair in San Francisco.
