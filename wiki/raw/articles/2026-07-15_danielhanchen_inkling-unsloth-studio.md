# Inkling added to Unsloth Studio

Daniel Han (@danielhanchen) announced that Thinking Machines' Inkling model has been added to Unsloth Studio, providing a unified local interface for the multimodal model.

## Key Features Added

1. **Interleaved tool calling support**: Inkling can now use tools within Unsloth Studio's framework
2. **Web search + code execution**: Built-in capabilities accessible through Studio
3. **Reasoning levels from 0.0 to 0.99**: Numerical reasoning control (Inkling natively uses numerical reasoning levels)
4. **Vision + Audio support**: Full multimodal capabilities accessible locally
5. **Full local execution**: Run Inkling on Mac, Windows, or Linux via Unsloth Studio
