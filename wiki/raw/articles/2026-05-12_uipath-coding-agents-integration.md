# UiPath for Coding Agents announcement

UiPath (NYSE: PATH) announced platform-wide integration for coding agents, enabling enterprise deployment of AI-generated automations.

Key capabilities:
- Open platform supporting any coding agent (Claude Code, OpenAI Codex, and future agents)
- Orchestration layer connecting agents with observability, execution, and governance
- Built-in governance: policy enforcement, audit trails, credential vaults, RBAC
- Model-agnostic: platform value compounds as models improve

Quote from Daniel Dines (CEO & Founder):
"The emergence of coding agents signals a fundamental shift in the definition of a builder on our platform."
