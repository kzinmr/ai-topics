# Replicate — About

Replicate is a platform that brings AI to every software developer. Their thesis is that AI isn't inherently hard — developers just don't have the right tools and abstractions yet.

**Core vision**: Make AI models as easy to use as normal software. Import an image generator like an npm package. Customize a model as easily as forking on GitHub.

**Key products**:
- Model hosting and inference API for open-source models
- Cog — containerization format for ML models
- Serverless GPU infrastructure
- Model fine-tuning capabilities

**Company**: Founded in 2019. Backed by a16z, Sequoia, and others. Based in San Francisco.

See https://replicate.com/blog for latest updates.
