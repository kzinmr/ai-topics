# Agreement is a Bug — Forcing Claude Code Agents to Disagree

**Bookmarks:** 1,128 | **Impressions:** 243K

## Key Thesis

- "Agreement is a bug" — consensus among agents can be a failure mode
- Experiment forcing 11 Claude Code agents to disagree with each other
- Purpose: extract diverse perspectives, avoid groupthink in multi-agent systems
- Related to sycophancy problems in LLMs

## Significance
Addresses a critical problem in multi-agent systems: if all agents converge on the same answer, you lose the diversity benefit. This connects to research on RLM/GEPA which also uses diversity (Pareto frontier) as an optimization signal.
