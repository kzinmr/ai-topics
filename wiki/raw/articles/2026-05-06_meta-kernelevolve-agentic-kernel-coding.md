# KernelEvolve: Meta's Agentic Kernel Authoring System for Heterogeneous AI Infrastructure

**Source**: https://engineering.fb.com/2026/04/02/developer-tools/kernelevolve-how-metas-ranking-engineer-agent-optimizes-ai-infrastructure/
**Date**: 2026-05-06

## Key Points

- Agentic AI system for autonomous kernel generation and optimization
- Uses LLM Synthesizer + Tree Search (MCTS) + RAG + Automated Evaluation
- >60% inference throughput improvement on NVIDIA GPUs for Andromeda Ads Model
- >25% training throughput improvement on Meta's MTIA custom silicon
- 100% pass rate on Stanford's KernelBench (250 problems)
- Solves cold start problem for proprietary hardware via systematic knowledge injection
- Paper: arXiv:25512.23236, to appear at ISCA 2026
