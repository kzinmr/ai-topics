# Agentic Engineering Patterns
**Source:** Simon Willison's Weblog | **Guide URL:** https://simonwillison.net/guides/agentic-engineering-patterns/

## Core Patterns Identified
1. **Tool-Use Agents** - Agents that can call external tools/APIs
2. **Multi-Agent Systems** - Multiple agents working together
3. **Agent-in-the-Loop** - Humans and agents collaborating
4. **Self-Improving Agents** - Agents that learn and adapt
5. **Specialized Agent Workflows** - Domain-specific agent patterns

## Key Concepts
- Agentic engineering focuses on building systems where AI agents can operate autonomously while maintaining human oversight
- Emphasis on practical implementation over theoretical frameworks
- Strong focus on evaluation and testing methodologies
- Integration with existing development workflows
