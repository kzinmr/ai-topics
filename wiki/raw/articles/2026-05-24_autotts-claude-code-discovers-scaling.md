# AutoTTS: Claude Code Discovers Test-Time Scaling Algorithms

## Source
The Decoder — May 24, 2026

## Key Points
- UMD/UVA/WUSTL/UNC/Google/Meta research
- Claude Code iteratively discovers test-time scaling control algorithms
- Achieves ~70% token reduction vs self-consistency at same accuracy
- Discovery cost: $40, 160 minutes
- Discovered algorithm tracks confidence shifts over rounds, not just final accuracy
- Transfers to different model and benchmark (DeepSeek-R1, GPQA-Diamond)
- Shifts human role from writing rules to designing search environments
