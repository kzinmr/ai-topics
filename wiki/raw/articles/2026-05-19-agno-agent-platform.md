# Agno: Agent Platform SDK & Runtime

40K+ stars, Apache 2.0 agent platform with three layers:
- **SDK**: Build agents, multi-agent teams, agentic workflows
- **Runtime**: Run agents as a production service (FastAPI, 50+ endpoints)
- **Control Plane**: Manage via AgentOS UI

Supports both Agno's own SDK and Claude Agent SDK. Provides sessions, memory, tracing, scheduling, RBAC, human approval flows. 

~30 lines of code gives you: FastAPI server, streaming (SSE/WebSockets), persistent sessions, per-user isolation, cron scheduling, OpenTelemetry tracing.

Latest: v2.6.5 (2026-05-06), 188 releases, 430 contributors.
