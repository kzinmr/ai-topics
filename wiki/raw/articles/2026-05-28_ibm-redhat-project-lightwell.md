# Project Lightwell: $5B Open Source Security Initiative

IBM and Red Hat announced Project Lightwell, a $5 billion commitment backed by frontier AI and 20,000+ engineers to secure enterprise open source software from upstream development through production.

## Model
- **Trusted Enterprise Clearinghouse**: AI-powered security coordination layer for vulnerability validation and patching
- **20,000+ engineers** augmented by AI for upstream maintenance, vulnerability review, patch development
- **Commercial subscriptions** with enterprise-grade validation and lifecycle management

## Early Adopters
Bank of America, BNY, Citi, Goldman Sachs, JPMorganChase, Mastercard, Morgan Stanley, Royal Bank of Canada, State Street, Visa, Wells Fargo

## Context
- Builds on IBM's 62,000+ OSS package footprint and expertise in 10,000+ technologies
- Incorporates learnings from Anthropic Project Glasswing and OpenAI Trust Access for Cyber
- Positions engineering capacity as premium strategic asset vs. AI-driven headcount reduction
