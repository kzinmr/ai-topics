## Overview

NVIDIA AI-Q is an open-source reference architecture (not a product) for building enterprise-grade deep research agents. It currently ranks #1 on both DeepResearch Bench I (55.95) and DeepResearch Bench II (54.50), beating OpenAI's closed system.

Built on NVIDIA NeMo Agent Toolkit and LangGraph, with LangChain Deep Agents library.

## Architecture

Two-tier routing: Intent Classifier → Shallow (fast, bounded) or Deep (multi-phase).

Deep Researcher orchestrated by LangGraph StateGraph with three specialized roles:
- **Planner**: Scout (maps info landscape) + Architect (designs evidence-grounded plan)
- **Researcher**: 5 specialist sub-agents in parallel (Evidence Gatherer, Mechanism Explorer, Comparator, Critic, Horizon Scanner)
- **Orchestrator**: Coordinates loop, fills gaps, synthesizes report

Key design decisions:
- Evidence-grounded planning: planner scouts before committing to structure
- Context isolation: each subagent works in own context window, returns only synthesis
- Configurable via YAML — swap models, tools, depth thresholds without code changes

## Model: Fine-Tuned Nemotron-3-Super

80k trajectories generated by GPT-OSS-120B as teacher → filtered by judge model (Qwen3-Nemotron-32B-GenRM-Principle) → 67k survived → fine-tuned Nemotron-3-Super-120B-A12B on 16 nodes × 8 H100 GPUs, ~25 hours, 5,615 steps.

Default config uses Nemotron-3-Nano (30B) + GPT-OSS-120B. Fine-tuned weights optional via Build API (limited availability).

## Middleware & Reliability

- Tool name sanitisation (alias resolution, fuzzy matching)
- Reasoning-aware retry (detect when model outputs only thinking tokens)
- Citation verification: deterministic post-processing verifies all citations against SourceRegistry using 5-level URL matching

## Enterprise Integration

- MCP server support for enterprise data sources
- Docker Compose and Helm charts (laptop to air-gapped data center)
- SKILL.md integration with Claude Code, Codex, Hermes Agent harnesses
- Validated on Dell AI Factory
