# The Agentic Web: How AI Systems Will Change Websites

**Source:** https://ricmac.org/2026/04/07/the-agentic-web/
**Published:** April 7, 2026
**Author:** Richard MacManus

## Core Thesis
The web is transitioning from a human-centric read/write platform to an **Agentic Web**, where AI systems interact with websites on behalf of users. This is an evolution of Tim Berners-Lee's vision.

## 4 Key Driving Forces

### 1. Websites Exposing Capabilities
- **MCP** (Anthropic Nov 2024): leading method for AI to trigger external actions
- **WebMCP**: Brings MCP tools directly to browser (requires extensions, will become native)
- **Custom chatbots**: Site-specific AI assistants becoming standard
- **Economic layer**: Capabilities shift from info retrieval to transactional execution

### 2. User Interfaces Are Changing
- Early AI: dumb headless browsers + brute-force scraping/form-filling
- Current/Future: guided by site owners, baked into browsers, powered by advanced tools
- **Discovery shift**: navigation via links → natural language prompts
- **SEO evolution**: traditional SEO → "AI retrieval optimization"

### 3. Browser Evolving into AI Runtime
- On-device AI: SLMs like Gemini Nano enable private low-latency AI in browser
- Interaction modes: Browsing Mode (human-like) vs Tool Mode (structured capabilities)
- Clear shift from browsing to tool-based interactions

### 4. Developer Platforms Adapting
- Cloudflare, Vercel, Netlify explosive growth; Vercel hosts agents via Next.js + MCP endpoints
- Standardized Agent UIs: MCP-UI, OpenAI Apps SDK & AgentKit, MCP Apps, Google A2UI
- UI as debug layer: transactional services may drop human UIs entirely

## Implications by Role
- **Product Teams**: Design for agents + humans, expose machine-legible capabilities, integrate Agent Experience (AX) alongside UX
- **Publishers**: Threat of fewer direct visits vs opportunity to be authoritative source of truth; compensation crisis
- **Developers**: Build machine-readable systems, think in orchestration, adopt hybrid stack (browser AI + cloud inference + MCP)

## Key Excerpt
> "You as the website owner are still in control of how AI systems — especially agents — can use your site."
