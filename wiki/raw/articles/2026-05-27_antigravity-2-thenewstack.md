# At Google I/O 2026: Antigravity Gets a New Job Description

Google is repositioning Antigravity from a coding environment into a platform for developing and managing teams of autonomous AI agents, backed by a new desktop app (2.0), CLI, SDK, and cloud integration.

## Antigravity 2.0 Platform
1. Desktop App: Central hub for orchestrating multiple agents simultaneously
2. CLI: Lightweight terminal interface, Gemini CLI users migrating
3. SDK: Programmatic access to agent harness, co-optimized for Gemini 3.5 Flash

## Managed Agents (Gemini API)
Single API call provisions full agent with isolated Linux environment. State/files persist across multi-turn sessions.

## CodeMender
AI code security agent that autonomously identifies vulnerabilities, recommends fixes, tests them, and applies patches. Integrated into Agent Platform as part of AI Threat Defense.

## Enterprise Adoption
- AirAsia Next: 50%+ production code generated via Antigravity
- Deloitte: "Governed, autonomous engineering workflows"
- PwC: "Shift from simple AI code completion to true agent orchestration"
- WPP: Integrated into WPP Open for product development
