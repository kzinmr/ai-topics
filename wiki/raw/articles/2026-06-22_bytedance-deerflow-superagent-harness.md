# DeerFlow 2.0 — ByteDance Open-Source SuperAgent Harness

## Overview

DeerFlow (**D**eep **E**xploration and **E**fficient **R**esearch **Flow**) is an open-source super agent harness that orchestrates sub-agents, memory, and sandboxes. Version 2.0 is a ground-up rewrite, sharing no code with v1. Claimed #1 on GitHub Trending on February 28, 2026.

**Key features:**
- Long-horizon SuperAgent harness that researches, codes, and creates
- Sandboxes, memories, tools, skills, subagents, and message gateway
- Handles tasks from minutes to hours
- Powered by extensible skills
