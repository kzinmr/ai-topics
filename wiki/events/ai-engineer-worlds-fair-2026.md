# AI Engineer World's Fair 2026

## Overview

The AI Engineer World's Fair 2026 (AIEWF) was held in San Francisco, marking three years since swyx coined the term 'AI Engineer' at the first AI Engineer Summit. The conference showed the field had matured from proving LLMs could act as autonomous agents (2023's focus) to building reliable production systems, orchestrating agent teams, and integrating AI into enterprise software.

## Key Themes

- **Shift from Agent Autonomy to Harness Engineering**: Contrast between Lilian Weng's 2023 and 2026 essays
- **Loop Engineering**: Inner/outer loop separation became the dominant design pattern
- **Forward Deployed Engineers**: New role for enterprise AI implementation
- **Software Factories**: Long-running agents managing the full software lifecycle
- **Context Engineering**: Enterprise data integration via MCP and APIs
- **Skill Engineering**: Markdown-based agent skills as a new paradigm
