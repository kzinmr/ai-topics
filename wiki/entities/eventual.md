# Eventual

Eventual is an AI data infrastructure company building Daft, an open-source engine for processing multimodal data (images, video, audio, text) at scale. It enables AI engineers to query petabytes of unstructured data as simply as SQL.

## Key Facts
- Founded in 2022; Y Combinator Winter 2022 batch.
- Sammy Sidhu's background spans HPC, deep learning, high-frequency trading, medical AI, and self-driving (DeepScale/Tesla, Lyft Level 5).
- Raised $30M across 3 rounds including Series A led by Felicis and Seed led by CRV.
- Strategic investors include Microsoft (M12) and Citi Bank.
- Used in production by Amazon, CloudKitchens, Together AI, Essential AI, and Fortune 25 companies.

## Products & Technology
- **Daft**: Open-source distributed data engine for multimodal workloads (images, video, audio, text, PDFs, 3D).
- 500x memory improvements and 15.7x throughput gains for multimodal data over traditional engines like Spark.
- Declarative DataFrame API with Rust-powered execution.
- 10x performance improvements over existing solutions for multimodal workloads.
