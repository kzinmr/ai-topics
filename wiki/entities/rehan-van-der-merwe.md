# Rehan van der Merwe (@rehanvdm)

## Overview

Software architect and blogger focused on AWS, serverless architecture, and practical system design. Author of articles challenging conventional cloud architecture wisdom.

## Key Contributions

- **Lambda Monolith (Lambdalith) Pattern:** Advocates for single Lambda functions handling all API routes via catch-all proxy, rather than per-route functions
- **Microservices vs Monolith Analysis:** Practical decision framework emphasizing team structure and Conway's Law
