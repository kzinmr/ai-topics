# UltraWorkers

UltraWorkers (ULW) is the GitHub organization that hosts the canonical **[[claw-code]]** repository. The organization was created to house the Rust implementation of the `claw` CLI agent harness, which is the primary product surface of the Claw Code project.

## Repository

- **claw-code** (https://github.com/ultraworkers/claw-code) — 186K+ stars, the fastest repo in GitHub history to surpass 100K stars
- **Primary language**: Rust (96.5%)
- **Focus**: Build-from-source Rust implementation; canonical source of truth for the `claw` CLI binary

## Relationship to Ecosystem

The UltraWorkers organization serves as the official home for the Rust port of Claw Code, while related ecosystem projects (oh-my-codex, oh-my-claudecode, clawhip) remain under individual developer accounts (Yeachan-Heo, code-yeongyu) or separate organizations (instructkr).
