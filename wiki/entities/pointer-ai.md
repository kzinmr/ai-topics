# Pointer AI

**Pointer AI** is an AI company focused on computer use agents — AI systems that operate computers like humans do. They achieved the highest verified scores on the [[concepts/computer-use|OSWorld]] benchmark.

## Key Achievement (May 26, 2026)

- **83.6% on OSWorld** with Claude Opus 4.7 (human baseline: 72.4%)
- **81.5% on OSWorld** with Claude Sonnet 4.6
- Top scores in 7 of 10 task domains
- Fully open source system release

## Architecture

- Lightweight task controller + 3 specialized agents
- Swappable model executor (Opus 4.7 or Sonnet 4.6)
- Different models excel at different domains
- Multi-apps category strongest differentiator

## Hiring Focus
- Computer use
- Verifying agent work
- Self-improving systems
- Coherence over long horizons
