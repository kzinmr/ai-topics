**The Open Stack for Self-Improving Agents** — train, deploy, and continuously improve AI models on an integrated stack for compute, RL post-training, environments, evals, and inference.

## Products & Platform

### Lab
Post-training platform for building self-improving agents via [[reinforcement-learning]]. Auto-instruments coding agents (e.g., [[openai-codex-cli]], [[anthropic-claude-code]]) for RL fine-tuning. Includes hosted evaluations with 100+ open-source models and a public leaderboard. Key customers: Ramp, Zapier.

### Environment Hub
2,500+ open-source RL environments contributed by the community. Environments are the core primitive — they define tasks, verifiers, and reward signals for agent training.

### Compute
Decentralized GPU compute marketplace enabling distributed training across global resources. Supports pipeline-parallel decentralized inference.

## Design Philosophy

Prime Intellect's technical philosophy, articulated through the `renderers` library and RL stack:

1. **For RL, the inference server should be a simple token-only endpoint** — Token-In, Token-Out
2. **Environments should be oblivious to tokenizers** — usable as evals for arbitrary model APIs
3. **Every assumption about chat templates will be violated eventually** — plan for it
4. **Official chat templates are often "wrong"** — require explicit opt-in repair
