# Thariq Shihipar

Thariq Shihipar is a member of technical staff at Anthropic working on [[concepts/claude-code|Claude Code]]. He co-authored the announcement of [[concepts/dynamic-workflows|dynamic workflows in Claude Code]] with Sid Bidasaria (@sidbid).

## Background

- **Current role**: Member of Technical Staff at Anthropic, working on Claude Code
- **Previous**: YC W20 (Y Combinator Winter 2020), South Park Commons, MIT Media Lab
- **X/Twitter**: [@trq212](https://x.com/trq212) (account since 2011)

## Notable Contributions

### Dynamic Workflows in Claude Code (2026-06-02)
Co-authored the article "A harness for every task: dynamic workflows in Claude Code" introducing the capability for Claude to dynamically generate custom agent harnesses at runtime. The article covers workflow patterns (classify-and-act, fan-out-and-synthesize, adversarial verification, tournament, loop-until-done), use cases (migrations, deep research, triage, sorting), and integration with Claude Code features like `/goal` and `/loop`.
