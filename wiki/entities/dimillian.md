# Guillaume Charpiat (@dimillian)

**Guillaume Charpiat** (@dimillian) is a software engineer working on AI-assisted development workflows. Known for practical experiments with coding agents, particularly OpenAI's Codex CLI.

## Notable Contributions

### Codex Mobile Power User Guide (June 2026)

Published a guide on using Codex Mobile as a **control plane** for development work running on remote machines (Mac/Windows/devbox). Key workflow:

1. Start threads on the right host + workspace from mobile
2. Pick branch + worktree from mobile
3. Run local environment setup before the task starts
4. Execute tasks remotely while mobile

This represents a **distributed coding agent pattern** — using the phone as an orchestration interface while the actual compute runs on dedicated hardware.
