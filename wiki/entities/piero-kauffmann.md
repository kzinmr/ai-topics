# Piero Kauffmann

Researcher at **Microsoft Research, AI Frontiers** lab. Focused on reinforcement learning infrastructure for AI agents. Built and maintained the RL infrastructure used for [[echo-rl|ECHO]] experiments.

- **Affiliation**: Microsoft Research AI Frontiers
- **Role**: RL infrastructure researcher

## Research Focus

- Reinforcement learning infrastructure and scaling
- GPU cluster management for agent RL training
- Multi-turn terminal agent environments
- Docker-based agent harnesses (Harbor)

## Notable Work

### ECHO: Terminal Agents Learn World Models for Free (2026-05)
Built and maintained the RL infrastructure used for all ECHO experiments, including the Harbor-based Docker environments, multi-turn rollout harness, and 8×B200 GPU training setup. Co-authored with [[entities/vaishnavi-shrivastava|Vaishnavi Shrivastava]], [[entities/dimitris-papailiopoulos|Dimitris Papailiopoulos]], and [[entities/ahmed-awadallah|Ahmed Awadallah]].

Paper: [arxiv.org/abs/2605.24517](https://arxiv.org/abs/2605.24517). Code: [github.com/microsoft/echo-rl](https://github.com/microsoft/echo-rl).
