# Kore.ai

Enterprise AI agent platform company serving 500+ Global 2000 organizations with 40+ voice/digital channels and 300+ integrations (Microsoft A365, Salesforce, HubSpot, Jira, GitHub, plus sector-specific systems).

## Artemis Agent Platform (May 2026)

Launched May 26, 2026 on Microsoft Azure. Designed for enterprises moving AI agents from pilots to daily operations in regulated sectors (banking, healthcare, insurance).

### Three Core Components

#### 1. Agent Blueprint Language (ABL)
A **compiled declarative language** for defining, validating, and governing AI agents, systems, and workflows. Six orchestration patterns:
- Supervisor, Delegation, Handoff, Fan-out, Escalation, Agent-to-agent federation

#### 2. Arch (AI Agent Architect)
Converts business objectives into production-ready ABL. Supports full agent lifecycle, topology design, and refinement using production traces.

#### 3. Dual-Brain Architecture
Two cognitive engines run in parallel via shared memory under a single runtime. Combines agentic reasoning with deterministic workflows for predictability and auditability.

### Governance as Architecture
> "Governance is architectural, not an afterthought." — Arunkumar Ramakrishnan, BCBS Massachusetts

- Every decision, path, and outcome is logged, traced, and analyzable in real-time
- Real-time PII tokenization, tenant isolation, immutable audit trails
- Compliance: SOC 2 Type II, ISO 27001, PCI DSS, FedRAMP Moderate, HIPAA, HiTrust, GDPR
- Deployment: public/sovereign/private cloud, on-premises with regional data residency

### Azure Integration
Built on Azure stack (compute, identity, AI, security) with native integration: Microsoft Foundry, Microsoft Agent 365, Entra ID, Microsoft Graph API, Azure Bot Framework (Teams channel).
