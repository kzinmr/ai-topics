# Claude Orbit

Anthropic's proactive AI assistant, leaked on May 5, 2026 inside Claude Cowork. Auto-generates briefings and insights from connected services without requiring user prompting.

## What It Does
- **Proactive insights**: Auto-generates briefings from Gmail, Slack, GitHub, Calendar, Drive, and Figma
- **Zero prompting required**: Surfaces relevant information before the user asks
- **Orbit apps**: Users can deploy and pin quick-access applications within the Orbit interface
- **Positioning**: Anthropic's answer to proactive assistant features from competitors

## Origins in Claude Code Leak

The March 31, 2026 Claude Code source leak revealed four unreleased feature flags that likely power Orbit:

| Codename | Description | Orbit Manifestation |
|---|---|---|
| **KAIROS** | Always-on daemon running 24/7, acts on user's behalf | Background briefing generation |
| **DREAM** | Nightly memory consolidation, reorganizes agent knowledge | Personalized insight synthesis |
| **ULTRAPLAN** | Offloads deep planning to remote Opus 4.6 (up to 30 min) | Deep analysis of user data |
| **COORDINATOR_MODE** | Multi-agent swarm with structured phases | Parallel processing of multiple data sources |
