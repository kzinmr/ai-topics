## Key Contributions

### The Causal Backbone Conjecture (2024)

Proposed a **selection theorem for agent-like structures** based on resource constraints rather than task richness. Key arguments:

1. **Against exhaustive world modeling**: Challenges *The Gooder Regulator Theorem* assumption that agents must model "sufficiently rich task sets" — argues this leads to **information overload**
2. **Universal causal backbone**: The universe has a core network of high-impact causal pathways defined by dominant resource flows
3. **Long-tailed influence distribution**: A small subset of entities (major corporations, key individuals, planetary systems) wield disproportionate causal power
4. **Sparse, high-leverage modeling**: Agents should track only backbone segments directly relevant to survival, not entire environments
5. **Resource distribution shifts**: Periodic massive redistributions create existential pressure — entities anchored to outdated distributions face extinction without adaptive capacity
