## Key Research Contributions

### Sparse Signal Loop (April 2026)

The **[[concepts/sparse-signal-loop|Sparse Signal Loop]]** experiment is stochi's most significant published work. It systematically tests the [[concepts/mismanaged-geniuses-hypothesis|Mismanaged Geniuses Hypothesis]] through a controlled 2×2 matrix across LongBench-Pro and Mini SWE Agent Plus benchmarks, investigating feedback density, memory location, and procedure persistence.

Key findings:
- Feedback sparsity is task-dependent, not universal
- The "Judge Gap" — models learned to satisfy the judge rather than solve the task (0.9667 Judge YES rate vs 0.5667 actual solve rate)
- Short, local, constrained memory outperformed free-form persistent memory
- More scaffold layers do not automatically unlock latent intelligence
