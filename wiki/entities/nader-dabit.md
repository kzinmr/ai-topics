# Nader Dabit (@dabit3) — Cognition Growth Engineer

Growth Engineer at Cognition Labs, extending Devin's capabilities beyond the terminal/IDE.

## Core Philosophy

### Agentic Slack

Nader's insight: agents don't need to live in terminals or IDEs.

- **Slack as an agent interface** — agents operate asynchronously in team communication channels
- Fire-and-forget task delegation via Slack threads
- Human-AI collaboration happens in the same space as human-human collaboration
- Eliminates the "switch context" problem of traditional agent interfaces

### Agents From Anywhere

Nader advocates for mobile-first, cross-platform agent orchestration:

- The future of agent interaction is not tied to a specific workspace
- Cross-platform access: terminal, IDE, Slack, mobile
- Democratizes access to AI-powered development
- **Engineering for agents that never sleep** — agents operate 24/7, humans review and direct

> "We're moving from 'agents at your desk' to 'agents everywhere'."

## Key Contributions

1. **Agentic Slack** — Integrating Devin into team communication workflows
2. **Mobile-first orchestration** — Managing agents from any device
3. **Asynchronous development** — Fire-and-forget task delegation
4. **Cross-platform access** — Breaking the terminal/IDE boundary
