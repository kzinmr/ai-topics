# Exa

**Exa** is an AI-native search platform positioning itself as a "search lab for agents" — providing search infrastructure designed specifically for AI agents rather than human users.

| | |
|---|---|
| **Type** | AI Search Infrastructure |
| **Founded** | ~2023 |
| **Key Products** | Agentic search platform, embeddings, web index |
| **Valuation** | $2.2B (May 2026) |

## Key Facts

- Raised **$250M Series C at $2.2B valuation** (May 2026)
- Positions as **search lab for AI agents** — search infrastructure purpose-built for agentic workloads
- Competes in the growing AI-native search infrastructure space alongside turbopuffer and Weaviate
