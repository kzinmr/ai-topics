# Claris (FileMaker) Agentic Coding Strategy

## Overview

Claris, an Apple company, announced in May 2026 a strategy to make FileMaker a first-class development target inside agentic coding tools (Claude Code, Cursor, Codex). This represents a significant shift in how low-code/enterprise platforms are adapting to AI-assisted development.

## Key Announcements

### FileMaker as Agentic Coding Target
- AI coding agents will understand FileMaker's schema, scripting language, and calculation engine
- Developers can describe what they need in natural language; agents write production-quality scripts directly into FileMaker solutions
- Results deploy directly into existing FileMaker infrastructure with automatic security and permissions inheritance
- Vendor-agnostic approach: supports multiple AI coding tools rather than locking into one

### Two Investment Areas
1. **Modern interfaces**: Web-native UI built through conversation with AI coding environments, rendered inside FileMaker
2. **Scripts and schema**: AI agents that understand file structure and FileMaker's scripting language, writing native scripts and custom functions

### FileMaker 2026
- Focus on resiliency, productivity, and infrastructure
- New features: FileMaker Server Remote Backup and Standby Server
- Lays groundwork for agentic development
- Developer previews of agentic coding functionality coming summer 2026

### Independent Shipping Model
- Improvements ship independently of traditional FileMaker release cycles
- More open development: preview things sooner, build in the open
- Community can extend skills alongside the platform evolution

## Strategic Significance

This is notable because:
- FileMaker has been a closed low-code platform for 40+ years
- Opening it to external AI coding agents (Claude Code, Cursor, Codex) rather than building a proprietary AI solution
- Acknowledges that "code is abundant" — the scarce resource is people who understand context and business requirements
- Represents how traditional enterprise software platforms are adapting to the agentic coding era

## Leadership

Ryan McCann, CEO of Claris (Apple company), authored the announcement.
