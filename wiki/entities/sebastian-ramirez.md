# Sebastián Ramírez

**Sebastián Ramírez** (handles: `tiangolo`) is a Colombian software developer and the creator of **FastAPI**, the most popular modern Python web framework for building APIs. He is also the creator of **Typer**, **SQLModel**, **Asyncer**, and a comprehensive ecosystem of Docker images for Python web deployment. He was the first recipient of the **Sequoia Open Source Fellowship** (2023), enabling him to work full-time on his open-source ecosystem.

## Overview

Sebastián Ramírez was homeschooled on a farm in rural Colombia, where he learned practical skills (planting, building, electrical work) that fostered a hands-on, inquisitive approach to problem-solving. He taught himself English and web development (HTML, CSS, JavaScript) using a hacked PalmPilot while roaming the mountains. At age 20, he survived a cancerous tumor in his chest — an experience that shifted his focus toward building tools with significant societal impact.

He has built APIs and tools for Machine Learning and data systems across Latin America, the Middle East, and Europe, working with teams at Explosion AI, Forethought, Anyscale, and others. He now lives in Berlin, Germany, where he works full-time on the FastAPI ecosystem and building **FastAPI Cloud**.

He did not attend university — he was homeschooled his entire life and is a self-taught developer who took online courses extensively.

## Core Projects

### FastAPI (2018–Present)
- **GitHub**: [fastapi/fastapi](https://github.com/fastapi/fastapi) (97.8k ★, 9.2k forks)
- Released Christmas Eve 2018
- Downloaded ~800,000 times per day (as of 2023)
- Used by: Uber, Netflix, Microsoft, Google, Space Telescope Science Institute (Webb Space Telescope data)
- Built on Starlette (ASGI) and Pydantic (data validation via Python type hints)
- Key features: automatic OpenAPI/Swagger docs, async support, dependency injection, type-hint-based validation

### Typer
- **GitHub**: [fastapi/typer](https://github.com/fastapi/typer) (19.3k ★)
- CLI library based on Python type hints — "the FastAPI of CLIs"
- 290M+ monthly downloads on PyPI

### SQLModel
- **GitHub**: [fastapi/sqlmodel](https://github.com/fastapi/sqlmodel) (17.9k ★)
- SQL databases in Python, designed for simplicity, compatibility, and robustness
- Combines SQLAlchemy and Pydantic

### Asyncer
- **GitHub**: [fastapi/asyncer](https://github.com/fastapi/asyncer) (2.4k ★)
- Async/await utility focused on developer experience

### Full Stack FastAPI Template
- **GitHub**: [fastapi/full-stack-fastapi-template](https://github.com/fastapi/full-stack-fastapi-template) (43k ★)
- Modern template: FastAPI, React, SQLModel, PostgreSQL, Docker, GitHub Actions, automatic HTTPS

### Docker Images
- **uvicorn-gunicorn-fastapi-docker** (2.8k ★) — High-performance FastAPI Docker image with auto-tuning
- **uwsgi-nginx-flask-docker** (3k ★) — Single-container Flask solution
- **nginx-rtmp-docker** (1.3k ★) — Live video streaming
- **dockerswarm.rocks** (1.1k ★) — Docker Swarm cluster guides

### FastAPI Cloud
- [fastapicloud.com](https://fastapicloud.com)
- Deployment platform built by Ramírez and the FastAPI team
- Primary sponsor and funding provider for FastAPI and friends open source projects

## Philosophy & Writing Style

Ramírez's development philosophy centers on **optimization, accessibility, and solving real problems**:

- *"If I feel I'm doing something simple, mechanical and repetitive, I just want to make it faster, or automate it."*
- *"I wanted things to be explained the way I wished they were explained to me when I was starting."*
- *"Innovation is merely the side effect or result of solving people's actual problems."*

His documentation is renowned for its clarity — he includes deep conceptual explanations and custom graphics, making complex topics accessible to developers of all levels. The FastAPI documentation is widely considered a gold standard for open-source project documentation.
