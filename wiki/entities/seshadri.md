# Seshadri (@sseshadri43)

**Seshadri** (handle `@sseshadri43`) is a software engineer at [[entities/langchain|LangChain]], working on Deep Agents infrastructure. They co-authored RubricMiddleware, a LangChain Deep Agents middleware that adds self-correcting grader loops to autonomous agents.
