# Ben Boyter

**Ben E. C. Boyter** (@boyter) is an Australian software engineer and creator of [[searchcode-com|searchcode.com]], [scc](https://github.com/boyter/scc) (Sloc Cloc and Code), and [cs](https://github.com/boyter/cs) (Code Spelunker). He is the author of the **"Marketing to the Machine"** thesis on Business-to-Agent (B2A) — designing services primarily for LLM consumption.

## About

Professional software engineer with 15+ years experience in development and testing. Has worked on:
- Internal web applications supporting thousands of users
- Customer-facing websites with 10s of millions of uniques/day
- Large financial applications handling multi-billion dollar projects
- HR applications for 10s of thousands of employees
- Large-scale AWS deployment and management
- Metadata Content Archive for 4,500 TB of video/audio with transcoding (2019)

## Key Projects

### searchcode.com
Source code search engine that indexed 75B+ lines of code, rebooted in 2026 as a **code intelligence MCP server for LLMs**. First website with LLM testimonials and a B2A (Business-to-Agent) design philosophy. See [[searchcode-com]] for full details.

### scc (Sloc Cloc and Code)
Ultra-fast code counter with complexity calculations. Go-based, supports 300+ languages, one of the most used code counters in the world. [github.com/boyter/scc](https://github.com/boyter/scc)

### cs (Code Spelunker)
Code search CLI with smart ranking, regex, and structural filters. Go-based. [github.com/boyter/cs](https://github.com/boyter/cs)

## B2A Thesis (March 2026)

In his blog post ["Marketing to the Machine"](https://boyter.org/posts/searchcode-has-been-rebooted/), Boyter articulated the shift from B2C to **B2A (Business-to-Agent)**:

> "In 2026, the average user of your site isn't 'Dave' filling in a form and clicking a mouse anymore. Half the internet is bot traffic... the human interface to the computer was changed. Forever."

Key points:
- **UI is secondary**: A website's visual interface matters less than the raw value it provides to agents
- **Efficiency = Brand Loyalty**: LLMs choose tools based on "Context ROI" — most info for fewest tokens
- **World first LLM testimonials**: searchcode.com features quotes from Claude Opus 4.6, Qwen 3.5, and Gemini as social proof for AI
- **Token economics**: searchcode claims 99% token reduction (267 MB raw code → 0.8 MB structured data)
