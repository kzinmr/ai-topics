# Shopify

Shopify is an e-commerce platform and a reference case for organizational AI adoption.

## AI Adoption Strategy (Under Tobi Lütke)

### Key Policies
1. **Reflexive AI Baseline:** "Reflexive AI usage is now a baseline expectation" — managers must prove AI can't do a job before hiring
2. **Unlimited AI Spend:** No quota on AI tool spending; token usage leaderboard
3. **Default Yes with Legal:** "If you don't default to 'yes,' you're defaulting to 'no'"
4. **Beginner-First Hiring:** Value adaptability over entrenched expertise

### Implementation
- 3,000 Cursor licenses deployed
- 80% GitHub Copilot adoption rate
- Fastest-growing AI users: support and revenue teams
- Legal team collaboration from the start

### Results
- Case study documented by First Round Capital
- 6,482 bookmarks, 2.50M impressions on announcement
- Referenced as a model for [[reflexive-ai]] implementation
