## Core Ideas

- **Filesystem as Memory**: Prefers file-based memory (CLAUDE.md, `.agent/` directories) over proprietary vector databases. Git provides native versioning, diffing, and branching.
- **Cache-First Engineering**: Every design decision in agent codebases should be evaluated through: "does this invalidate the prompt cache?"
- **Background Agents**: Production-ready for well-scoped async work. Real companies (Coinbase, Ramp, Stripe) getting measurable PR velocity.
- **Graph-Based Coordination**: Graphviz DOT syntax for workflow definition with branching, loops, and parallelism primitives.
