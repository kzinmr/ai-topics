# UiPath

**UiPath** (NYSE: PATH) is a global leader in agentic business orchestration and enterprise automation. Founded by Daniel Dines, the company provides a platform that combines robotic process automation (RPA) with AI agent orchestration, enabling enterprises to automate business processes at scale.

## Platform Capabilities

UiPath's platform provides:

- **Visual orchestration**: Drag-and-drop workflow design for business processes
- **Enterprise deployment**: Production-grade automation with monitoring and governance
- **Multi-agent coordination**: Orchestration layer connecting coding agents, RPA bots, and human workers
- **Built-in governance**: Policy enforcement, audit trails, credential vaults, role-based access control

## UiPath for Coding Agents (May 2026)

On May 12, 2026, UiPath announced **UiPath for Coding Agents** — a platform-wide integration enabling coding agents to become enterprise-deployable. Key differentiators:

| Feature | Detail |
|---------|--------|
| **Open platform** | Supports any coding agent — Claude Code, OpenAI Codex, and future agents |
| **Orchestration layer** | Constant across model versions — connects agents with observability, execution, and governance |
| **Built-in governance** | Policy enforcement, audit trails, credential vaults, RBAC, runtime controls |
| **Model-agnostic** | Platform value compounds as models improve; automations survive model replacement |

### Strategic Vision

Daniel Dines stated: *"The emergence of coding agents signals a fundamental shift in the definition of a builder on our platform. We are first to market with a platform that treats AI-generated automations as first-class citizens, with the same governance, reliability, and scale that enterprises demand."*

### Enterprise Impact

- Enables non-technical users (business analysts, process owners, domain experts) to create automations through natural language conversation with coding agents
- Removes manual handoffs between coding agent output and enterprise deployment pipelines
- Automations continue running when models are replaced or developers move on
- Initial support for [[concepts/claude-code]] and [[concepts/codex]], with more integrations planned for 2026

## Market Position

UiPath is positioning itself as the **orchestration layer** between coding agents and enterprise systems. Their thesis is that while individual coding agents (Claude Code, Codex, etc.) compete on model capability, the platform that connects agents to CI/CD infrastructure, testing frameworks, and governance controls becomes more valuable as more agents enter the market.

This is distinct from developer-focused coding agent platforms like [[entities/coder]] (self-hosted) or [[entities/anysphere]] (Cursor, developer IDE).
