# Letta (Letta AI)

> **Core Thesis:** Memory is the key to self-improving AI. Letta builds open, model-agnostic agent harnesses with persistent memory systems.

## Overview

Letta AI is a company focused on building memory systems for AI agents. Founded on the belief that memory was the key to self-improving artificial intelligence, Letta has pivoted from chat-based memory to building **Letta Code** — an open, model-agnostic agent harness.

## Letta Code

Letta Code is the flagship product: a **model-agnostic agent harness with persistent memory** that gives underlying LLMs frontier "agent" capabilities:

- **Computer Use** — Inject agents into environments where they can execute scripts and modify state
- **Skills** — Pre-built capability modules
- **Subagents** — Hierarchical agent delegation
- **Transparent Memory Systems** — Persistent, inspectable memory
- **Deployment Paths** — Production-ready deployment options

### Key Differentiator

Unlike closed labs (OpenAI Responses API, Anthropic Agent SDK, Google Gemini for Computer Use, Microsoft Copilot Studio), Letta Code is:

1. **Open** — Source-available
2. **Model-Agnostic** — Works with any LLM
3. **Not locked to a single provider** — Can swap underlying models
