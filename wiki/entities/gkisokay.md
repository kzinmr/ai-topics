# gkisokay

**gkisokay** (Graeme, @gkisokay) is an AI agent practitioner, founder of Amplifi (@amplifi_now), and a prominent advanced user in the Hermes Agent community. Known for his "Building AGI for my Hermes Agent" series and agent watchdog architecture patterns.

## Overview

Graeme runs Amplifi, a company focused on helping businesses scale with AI through agent-based solutions. On X, he is one of the most recognized advanced Hermes Agent users, regularly sharing deep technical patterns for agent architecture.

## Key Contributions

### Building AGI for Hermes Agent Series (April 2026)
A multi-part X thread series documenting progressive agent self-improvement:
- **Level System (Day 5)**: Built a 10-dimension intelligence scorecard (17/50) to benchmark agent progress toward AGI
- **Auto-think / Auto-build (Day 13)**: Engineered autonomous thinking and building capabilities into Hermes agents
- **Upstream Research Agent Pattern**: Developed an architecture where one upstream research agent feeds structured information to the entire agent system

### Core Thesis: Structured Research > Raw Scraping
Articulated a key principle in agent architecture: "raw scraping the web is not research. If the information is not structured, your other agents can't use it." This has become a recognized pattern in the Hermes Agent community as the **structured research pipeline** pattern.

### Agent Watchdog Architecture
Recommended real-time agent watchdog pattern: Codex + GPT-5.4 as a monitor on Hermes-driven workflows, live-coding fixes as drift appears (70+ likes on X). This pattern was documented in the Hermes Agent community analysis as a canonical advanced ops pattern.
