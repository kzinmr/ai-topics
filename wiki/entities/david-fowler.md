# David Fowler

David Fowler (@davidfowl) is a **Distinguished Engineer at Microsoft**, known for creating foundational .NET ecosystem projects and now working on AI-powered developer tooling with Aspire.

## Key Projects

- **NuGet** — .NET package manager (co-creator)
- **SignalR** — Real-time web functionality for .NET (creator, started as side project → became Microsoft product)
- **ASP.NET Core** — Cross-platform web framework (architect)
- **Aspire** — Open-source, local-first platform for orchestrating application stacks from code (frontends, APIs, containers, databases, cloud resources). Agent-ready and production-ready.
- **Tally** — AI-powered bank transaction classifier (personal project, 1.1K+ GitHub stars)

## Notable Writing

- **"AI Made Us Faster. That Was the Problem"** (May 2026) — Reflects on the Aspire team's experience: building production software with AI is fundamentally different from building demos. "It is one thing to use AI to generate a prototype, automate a small task, or explore an idea. It is another thing to use AI inside a real product team, on a real codebase, with real customers, docs, tests, releases, and maintenance costs."

## Philosophy

Fowler's work consistently bridges **developer tooling** and **production realities**. His perspective on AI coding emphasizes that speed gains don't automatically translate to better software — the production engineering discipline (testing, review, maintenance) becomes more critical, not less, when AI accelerates code generation.
