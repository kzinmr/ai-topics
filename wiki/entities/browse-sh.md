# Browse.sh

**Browse.sh** is an open catalog of 100+ curated browser skills that any agent can install with one CLI command. Built by [[entities/browserbase|Browserbase]] and launched May 2026, it captures how to navigate real websites so agents stop re-discovering every site from scratch on every run. Skills are durable, reusable playbooks — markdown files plus helper scripts — powered by **Autobrowse**, a system that uses AI to iterate on real tasks until it converges on the cheapest, fastest path.

Browse.sh addresses the "discovery tax" that browser agents pay: every run re-learns site structure, undocumented APIs, and gotchas that a previous run already solved. By encoding this knowledge as skills, Browse.sh reduces per-run costs by ~45% (benchmarked on Craigslist: $0.22 → $0.12).

## Getting Started

```bash
npm i -g browse
browse skills add zillow.com/extract-listings
browse skills list
```

## Key Features

### Agent Skills as Playbooks
Each skill is a `SKILL.md` file + optional helper scripts containing the exact steps, gotchas, API endpoints, selectors, and fallback strategies needed for a specific site. No vector embeddings — plain text that humans can read and agents can execute.

### Autobrowse: AI That Improves AI
Skills graduate from **Autobrowse**, a system where an agent runs a real task on a real site, studies its own trace, iterates on its strategy, and continues until the workflow converges on reliability. The agent then writes out a durable skill.

### Site-Specific Knowledge Encoding
Skills capture hard-won reverse-engineering knowledge:
- Undocumented JSON endpoints (e.g., `sapi.craigslist.org`)
- Positional array decoding quirks (`item[0]` is an offset, not a posting ID)
- IP-based geo-scoping workarounds (`postal=<zip>` override)
- Rate-limit boundaries (≤ 1 req/s sustained)

### Agent-Native Integration
Skills follow the **AgentSkills standard** (SKILL.md format), making them compatible with Claude Code, OpenAI Codex, and other major agent frameworks. The Browse CLI (`npm i -g browse`) lets agents discover and load skills on demand.


## Economics: The Discovery Tax

Browse.sh quantifies the "discovery tax" that agents pay on every new session:

| Metric | Generic Agent Loop | With Browse.sh Skill |
|--------|-------------------|---------------------|
| Cost per run (Craigslist) | ~$0.22 | ~$0.12 |
| Discovery turns | 30+ turns exploring | 0 (skill provides known path) |
| Cost reduction | — | **45%** |

At scale, this compounds: skills encode the shortest reliable path and reuse it instead of re-deriving it every run.
