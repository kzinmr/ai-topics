# Telegram

Telegram is a cloud-based messaging platform and bot ecosystem that supports extensive third-party integrations, AI-powered bots, Mini Apps (Web Apps), and managed bot infrastructure.

## Overview

Telegram operates as both a messaging service and a developer platform. Founded in 2013 by Pavel and Nikolai Durov, it has evolved into a significant deployment surface for AI systems.

- **Platform**: Cloud-based, cross-platform (iOS, Android, Desktop, Web)
- **Bot API**: Extensive API with local Bot API option (unlimited file transfers)
- **Mini Apps**: JavaScript-based applications running inside Telegram with 500M+ monthly users
- **Telegram Stars**: In-app currency for digital transactions
- **Bots for Business**: Telegram Business account integration

## Bot Platform Features
