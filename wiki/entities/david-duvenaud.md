# David Duvenaud

David Duvenaud is an AI researcher known for work on neural architecture search, Gaussian processes, and deep learning. In April 2026, he co-announced **Talkie**, an open-weight 13B LLM trained exclusively on pre-1930 data, alongside Alec Radford and @status_effects.

## Key Work
- **Talkie** (April 2026): Historical LLM trained on pre-1930 data only
- Neural architecture search and differentiable structure learning
- Gaussian process applications in deep learning
