# GPT-5.5

**GPT-5.5** is OpenAI's language model released in April 2026, representing a significant step forward in both performance and efficiency.

## Overview

GPT-5.5 was released in late April 2026 and quickly established itself as a leading frontier model. Key highlights:

- **Outperforms Opus 4.7** on paper across most benchmarks
- **40% more token efficient** compared to GPT-5.4
- **2x more expensive per token** than GPT-5.4, but per-task costs remain stable due to efficiency gains
- Aggressive rollout with frequent rate limit resets

### New Evaluation Frameworks

- **VibeBench**: Subjective evaluation framework using 1,000 software engineers to rate model quality through real-world usage
- **ParseBench**: Document intelligence benchmark for evaluating parsing capabilities

## Architecture & Features

- **Image-to-code workflow**: Use new image generation model to create designs, then GPT-5.5 converts to code
- **Agentic capabilities**: Strong performance on multi-step coding tasks
- **Rate limit resets**: OpenAI frequently adjusting limits during rollout
