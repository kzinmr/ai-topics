# @cyrilxbt

X/Twitter content creator focused on the intersection of **Obsidian**, **Claude Code**, and **personal knowledge management** (PKM). Produces long-form X Articles and threads on vault architecture, knowledge organization systems, and LLM-augmented retrieval workflows.

## Content Focus

- **Obsidian vault methodology**: Folder structures, naming conventions, YAML properties, tagging taxonomies, Maps of Content (MOCs)
- **Claude Code + Obsidian integration**: Using Claude Code with Filesystem [[mcp]] to query and navigate Obsidian vaults via natural language
- **Retrieval-first organization**: Designing knowledge systems optimized for finding information quickly rather than storing it neatly
- **Vault maintenance**: Inbox processing habits, quarterly vault reviews, progressive reorganization strategies

## Key Contributions

A comprehensive X Article covering the complete methodology for organizing Obsidian vaults around retrieval-first principles. ## Relevance to AI/LLM Domain

@cyrilxbt's content bridges personal knowledge management with LLM tooling. The Claude Code + Obsidian integration pattern they describe is an instance of the broader **LLM Wiki** concept (inspired by [[karpathy]]'s knowledge base workflow), where an LLM maintains a structured markdown wiki from raw sources.
