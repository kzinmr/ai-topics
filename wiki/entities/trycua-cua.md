# TryCua (Cua)

Open-source framework enabling AI agents to control full operating systems within high-performance, lightweight virtual containers. Backed by Y Combinator (X25 batch).

## Overview

Cua delivers up to 90% native speed on Apple Silicon and works with any language model. It provides the infrastructure layer for computer-use agents — giving agents a full OS environment to interact with, rather than just browser DOM access.

## Funding

- **Y Combinator X25** batch
- **Pre-seed**: $500K (Jul 2025)
- Company size: 2-10 employees
- Headquarters: San Francisco, California

## Technology

- Virtual containers for agent OS control
- Up to 90% native speed on Apple Silicon
- Language-model agnostic
- Open source (13.3k+ GitHub stars)
