## Overview

Will McGugan is one of the most creative and prolific Python developers of the modern era. His work is defined by a simple but powerful insight: **terminals are underutilized as interactive interfaces**, and with the right tools, they can rival graphical applications for many developer workflows.

His flagship project, **Rich** (2020), brought beautiful terminal formatting to Python with support for colors, styles, tables, markdown, syntax highlighting, progress bars, and more. With 49.6k+ GitHub stars, Rich has become one of the most widely used Python libraries for terminal output, adopted by projects ranging from CLI tools to Jupyter notebooks to production logging systems.

Building on Rich's success, McGugan created **Textual** (2022), a full-featured terminal UI framework that enables developers to build complex, interactive applications with widgets, layouts, event handling, and reactive programming — all in pure Python, running in any standard terminal. Textual has emerged as the leading framework for building terminal-based applications that were previously impossible or impractical.

In 2025, after his startup funding ran dry, McGugan channeled his skills into **Toad** (December 2025), a terminal-based front-end for AI coding agents. Toad supports 12 agent CLIs (OpenHands, Claude Code, Gemini CLI, and others) through the ACP protocol, providing a unified interface for agentic coding with features like fuzzy file search, markdown rendering, shell integration, and tab completion.

McGugan's development philosophy centers on **craftsmanship, user experience, and open source**. Every project is designed to be beautiful, functional, and accessible. His writing at willmcgugan.com combines technical depth with genuine enthusiasm for the possibilities of terminal-based computing.

## Core Ideas

### The Terminal Is Underrated as an Interactive Interface

McGugan's central thesis is that the terminal — often dismissed as a primitive text-only interface — is actually a powerful and underutilized platform for building rich, interactive applications:

> "I have acquired over a very long career convincing terminals they are actually GUIs."

His work on Rich and Textual demonstrates that terminals can support:
- **Rich formatting** — colors, styles, tables, markdown, syntax highlighting
- **Interactive widgets** — buttons, inputs, lists, trees, progress bars
- **Event-driven programming** — key presses, mouse clicks, focus changes
- **Reactive layouts** — dynamic UI that responds to data changes
- **Full-screen applications** — complex, multi-panel interfaces with navigation

This challenges the assumption that GUIs require graphical toolkits and that terminals are limited to text output. McGugan shows that with the right abstractions, terminals can be as expressive as any GUI framework — while remaining lightweight, portable, and composable.

### Open Source as a Viable Career

McGugan founded **Textualize** to build a sustainable business around open-source terminal tools. The company offers:
- Commercial support for Rich and Textual
- Consulting services for terminal application development
- Premium features and integrations
- Community building around the Python TUI ecosystem

His approach demonstrates that open-source software can be both a creative outlet and a viable career path — if you build something genuinely useful and find the right monetization model.

### AI in the Terminal: Toad's Philosophy

After his startup wrapped up in mid-2025, McGugan built **Toad** (a play on "Textual Code") as a front-end for AI coding agents. His philosophy:

> "Toad is a unified experience for AI in the terminal. I don't have money, but what I do have are a very particular set of skills."

Key design principles:
- **Protocol-agnostic** — supports any agent CLI through the ACP protocol (12 agents at launch)
- **Terminal-native** — full TUI with markdown rendering, fuzzy search, shell integration, tab completion
- **Developer-friendly** — preserves native terminal features (scrolling, search, mouse interaction)
- **Open-source** — built by McGugan with community contributions and sponsorship support

Toad represents McGugan's vision for the future of developer tooling: **AI agents that work in the environments developers already use, with interfaces that feel natural and powerful**.

### Markdown Rendering in Terminals

McGugan has invested significant effort in making terminal-based markdown rendering competitive with web-based alternatives. His approach:
- **Snappy differential rendering** — only redraw changed lines, not the entire screen
- **Full syntax highlighting** — code fences with proper language detection
- **Table support** — complex tables with alignment and borders
- **Markdown-as-you-type** — real-time preview while composing
- **SVG export** — ability to export terminal output as vector graphics

This work demonstrates that terminal interfaces can be as rich and expressive as any GUI, while remaining lightweight and portable.

## Key Work

### Rich (2020–present)
The most popular Python terminal formatting library with 49.6k+ GitHub stars. Features:
- **Colors and styles** — full terminal color support with style composition
- **Tables** — complex tables with borders, alignment, and formatting
- **Markdown** — render markdown documents with proper formatting
- **Syntax highlighting** — code blocks with language-specific highlighting
- **Progress bars** — animated, customizable progress indicators
- **Tracebacks** — beautiful, readable error output with context
- **Logging** — structured, colorful log output for debugging
- **Jupyter integration** — works in notebooks and interactive environments

### Textual (2022–present)
A full-featured terminal UI framework for Python with 25.8k+ GitHub stars. Features:
- **Widget system** — buttons, inputs, lists, trees, tables, and more
- **Layout engine** — CSS-like layouts with flex, grid, and custom arrangements
- **Event handling** — key presses, mouse clicks, focus changes, custom events
- **Reactive programming** — data binding, computed properties, watchers
- **Animation** — smooth transitions and effects
- **Cross-platform** — works on macOS, Linux, and Windows terminals
- **Extensible** — plugin architecture for custom widgets and integrations

### Toad (2025–present)
Terminal-based front-end for AI coding agents. Features:
- **Multi-agent support** — 12 agent CLIs through ACP protocol (Claude Code, OpenHands, Gemini CLI, etc.)
- **Fuzzy file search** — filters based on .gitignore patterns
- **Markdown rendering** — beautiful streaming output with syntax highlighting
- **Shell integration** — run commands with full color and interactivity
- **Tab completion** — familiar terminal completion for commands and paths
- **Conversation history** — cursor through previous interactions, copy to clipboard
- **SVG export** — save terminal output as vector graphics

### batrachian.ai (2025)
McGugan's personal AI-powered project website, built using Toad and his terminal tools. Demonstrates the practical application of his TUI philosophy for real-world projects.

### Textualize (Company)
The business entity supporting McGugan's open-source work:
- Commercial support for Rich and Textual
- Consulting and custom development
- Community building and education
- Sponsorship and funding coordination
