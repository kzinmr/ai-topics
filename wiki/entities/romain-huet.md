# Romain Huet

**Romain Huet** (X: @romainhuet) is Head of Developer Experience at **OpenAI** and host of the **Builders Unscripted** interview series. A French entrepreneur and engineer based in San Francisco, he previously led developer-facing products at Stripe (Developer Relations → Product for Developer Platform) and Twitter (Developer Relations). He was co-founder and CTO of Jolicloud.

## Bio

Romain Huet leads Developer Experience at OpenAI, focusing on helping developers adopt and build with OpenAI's tools — particularly **Codex** (the CLI coding agent) and the broader OpenAI API platform. His role bridges the gap between OpenAI's product teams and the developer community, surfacing builder needs and showcasing real-world use cases.

Huet created and hosts **[[concepts/builders-unscripted|Builders Unscripted]]**, an OpenAI-produced interview series that spotlights "the stories behind real projects and the mindset that makes them possible." The series tagline: *"You can just build things."*

## Key Activities at OpenAI

Beyond Builders Unscripted, Huet is a frequent presenter of OpenAI's developer tools:

### Codex CLI Launches & Demos
- **Codex CLI with GPT-5-Codex** (2026): Demoed multiplayer game building from scratch using natural language, including Vercel deployment — all without writing code.
- **Codex CLI Launch** (w/ Fouad Matin, Agents Research): Introduced Codex as an open-source local coding agent, demonstrated dark mode implementation, codebase explanation, and sandboxed execution (network-disabled by default in full-auto mode).
- **Automatic Code Reviews** (w/ Maja Trębacz): Showcased Codex code review for GitHub PRs and Codex CLI local `/review`, including internal OpenAI usage catching training-run bugs before production.

### Philosophy
- **Builder-first developer experience**: Focus on what developers actually need to build with AI, not just API documentation
- **Codex as teammate**: Emphasizes Codex as an AI teammate available at any surface (CLI, editor, cloud) rather than just a code generation tool
- **Code review as safety layer**: Advocates for automated code review as essential infrastructure that scales with AI capability improvements
- **Unscripted format**: Conversations that reveal the real stories and mindsets behind successful AI projects
