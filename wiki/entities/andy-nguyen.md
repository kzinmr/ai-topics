# Andy Nguyen

**Duy Anh "Andy" Nguyen** (@kevinnguyendn) is a Vietnamese entrepreneur and the Founder & CEO of ByteRover, a portable, file-based memory layer for autonomous coding agents. He is building what he calls "an agentic memory layer for coding agents to help millions of devs vibe code better."

## Overview

Andy Nguyen is based in Da Nang, Vietnam. He founded ByteRover in September 2024 after working as a Machine Learning Engineer at OpenLab JSC (2022-2024), an AI solutions provider for SMEs based in Da Nang. OpenLab developed Avoca AI, a no-code platform for creating and deploying customized AI models.

Before his ML career, he worked as a self-employed ML engineer from February 2024. He has skills across HTML5, CSS, JavaScript, Python, Ruby on Rails, SQL, UI/UX, and Data Science.

## ByteRover

Founded ByteRover in September 2024. The project started as **Cipher**, an open-source MCP-based memory layer for coding agents, and evolved into **ByteRover CLI (brv)**, a full-featured memory layer with:
- Git-like version control for context
- Hierarchical Context Tree architecture
- Market-best 92-96% retrieval accuracy on LoCoMo/LongMemEval
- 4.2k+ GitHub stars
- Native plugins for Hermes Agent, Claude Code, and OpenClaw
- arXiv paper (2604.01599)

His personal thesis: "Your engineering team is likely wasting 35-50% of LLM spend and stretching project cycle time unknowingly because your coding agents are running on noisy, unstructured context."
