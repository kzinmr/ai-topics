# Portkey

**Portkey** is an AI agent security platform that protects autonomous agents processing trillions of tokens monthly. In May 2026, [[entities/palo-alto-networks|Palo Alto Networks]] announced its acquisition of Portkey (deal closes Q4 2026).

## Core Capabilities

| Capability | Description |
|:---|:---|
| **Action Inspection** | Examines every AI agent action in real-time |
| **Risk Prevention** | Stops risky agent behavior before execution |
| **Traffic Monitoring** | Monitors all AI traffic across models |
| **Model Management** | Manages thousands of AI models |
| **Uptime** | 99.99% uptime guarantee for autonomous agents |

## Why It Matters

AI agents increasingly operate with special access to enterprise systems — acting like powerful employees. Without security, they become targets for attacks. Portkey addresses the gap between agent deployment velocity and security oversight, fitting into the broader [[concepts/shadow-ai-governance|Shadow AI Governance]] crisis where 80% of Fortune 500 companies have lost control of their AI infrastructure.

## Acquisition Context

Palo Alto Networks' acquisition of Portkey signals that AI agent security is maturing from a niche concern into a core enterprise cybersecurity category. This parallels the trajectory of:

- [[concepts/agent-iam|Agent IAM]] — identity and access management for non-human identities
- [[concepts/agentic-security|Agentic Security]] — the broader security patterns for AI agents
- [[concepts/agent-governance|Agent Governance]] — structured approaches to agent oversight
