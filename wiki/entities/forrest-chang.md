# Forrest Chang

Forrest Chang is the developer who created the [andrej-karpathy-skills](https://github.com/forrestchang/andrej-karpathy-skills) GitHub repository — the fastest-growing single-file repo of 2026. After reading Andrej Karpathy's January 2026 thread about LLM coding failure modes, Chang converted Karpathy's observations into 4 behavioral rules in a single `CLAUDE.md` file.

## Key Contribution

The `CLAUDE.md` file distilled Karpathy's complaints about AI coding agents into actionable, machine-readable guidelines:
1. Think Before Coding
2. Simplicity First
3. Stay in Scope
4. Goal-Driven Execution

The repo hit 5,828 stars in its first day, 60,000 bookmarks in two weeks, and 120,000+ stars by May 2026 — demonstrating that a 65-line markdown file with zero dependencies can be more impactful than many full frameworks. Released under the MIT license.
