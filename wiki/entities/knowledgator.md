# Knowledgator

**URL:** https://knowledgator.com
**HuggingFace:** https://huggingface.co/knowledgator
**GitHub:** https://github.com/Knowledgator
**Discord:** https://discord.gg/dkyeAgs9DG

Open-Source ML research company specializing in information extraction technology using encoder-based models. Known for developing and releasing the GLiNER family (NER, classification, relation extraction).

## Key Products

| Product | Domain | Foundation |
|---|---|---|
| **[[concepts/gliclass]]** | Zero-shot text classification | DeBERTa / ModernBERT |
| **GLiNER** | Zero-shot named entity recognition | Multi-task encoder |
| **GLiNER-bi-V2** | Bi-encoder NER | bi-encoder |
| **GLiNER-X** | Multilingual NER | mDeBERTa |
| **GLiNER-BioMed** | Medical NER | Domain-specific |
| **GLiNER-relex** | Relation extraction | GLiNER-derived |
| **GLiNER-Linker** | Entity linking | GLiNER-derived |
| **GLiNER-PII** | PII (Personal Identifiable Information) detection | GLiNER-derived |
| **ModernGLiNER** | ModernBERT-based NER | ModernBERT |
| **ModernGLiClass** | ModernBERT-based classification | ModernBERT |
| **LLM2Encoder** | Encoder distillation from LLMs | Distillation technique |
